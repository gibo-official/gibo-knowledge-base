## 2.1 20年发展历程

洁博利用20年时间，从福州一间厂房成长为国家级专精特新企业：

| 年份 | 里程碑事件 |
|:----:|-----------|
| **2005** | 创始人郑少波在上海成立研发工作室，深耕感应洁具核心电路研发 |
| **2008** | 落地福州——中国感应洁具发源地，通过ISO 9001认证 |
| **2009** | 与国际知名卫浴品牌建立深度ODM战略合作 |
| **2010** | 成为**上海世博会指定供应商**；通过欧盟CE、RoHS认证 |
| **2014** | 首次获评**国家高新技术企业** |
| **2015** | 行业首创双感应快装水嘴"小洗宝"，开创感应水嘴细分品类 |
| **2019** | 首获**沸腾质量金奖** |
| **2022** | 参编国家标准**GB/T 41863-2022** |
| **2023** | 获评**国家专精特新中小企业**；dTOF技术再获沸腾质量金奖 |
| **2024** | 参编团体标准**T/XMBK 002-2024** |
| **2025** | 累计产销突破**500万套**；推出4D奢享系列 |
| **2026** | 成立20周年——专注感应洁具ODM，产品出口40+国家 |
