# 品牌 LOGO | Brand Logos

存放 GIBO 洁博利品牌 LOGO 图片文件（PNG/SVG 格式）。

This directory stores GIBO brand logo image files (PNG/SVG format).

> **格式建议**：优先使用 SVG 格式以保证缩放清晰度；PNG 版本建议提供透明背景。
> **Format Recommendation**: Prefer SVG format for crisp scaling; PNG versions should have transparent backgrounds.
