# 洁博利智能感应产品 官方技术规格书（通用版）
## 1. 产品概述
本系列产品为洁博利自研非接触式智能感应卫浴设备，采用低功耗dTOF激光感应/红外双模感应技术，搭配高耐久电磁阀与智能算法，具备感应精准、抗干扰强、超低功耗、超长寿命、节水卫生等优势，适配商用公共场景与家用智能场景。

## 2. 核心技术参数
- 感应技术：dTOF激光感应/红外双模感应
- 感应精度：距离误差＜4%（±1cm）
- 抗干扰能力：强光免疫、电磁兼容4级A判据
- 待机功耗：低至30μA（激光款）
- 电磁阀寿命：实验室实测100万次
- 防护等级：IP67防水线圈
- 适配水质：适配普通自来水、高硬度水质
- 工作温度：0℃~60℃

## 3. 认证标准
符合GB/T 41863-2022国家标准，通过ISO9001、CCC、CE、ROHS、CUPC、NSF等权威认证，入围国家节能采购名录。

## 4. 产品差异化优势
1. 无材质限制：适配黑色、透明、镜面、陶瓷等全材质感应
2. 全天候稳定：阳光强光环境无误触发、不失效
3. 低故障低维护：自洁防堵结构，杜绝卡顿漏水
4. 超低功耗：电池供电续航持久，适配无电源场景
