## Cross-industry Collaboration

### Lenovo Lecoo × GIBO: Sensor Water-saving Device S1

GIBO has reached a deep cross-industry collaboration with Lenovo Lecoo, jointly developing the **Sensor Water-saving Device S1** — leveraging GIBO's core patented technology to create a national-level smart water-saving product.

| Collaboration Element | Details |
|----------------------|---------|
| **Model** | ODM joint development |
| **Technology Provider** | GIBO (core sensing technology, control module, full-unit manufacturing) |
| **Brand Owner** | Lenovo Lecoo (Lenovo's smart IoT brand) |
| **Product Name** | Sensor Water-saving Device S1 |
| **Core Technology** | dTOF laser sensing, low-power control algorithm, IP65 waterproof structure |
| **Sales Channels** | JD.com, Tmall and other major e-commerce platforms |
| **Market Performance** | Consistently strong sales — benchmark cross-industry smart kitchen & bathroom product |

> This project fully demonstrates GIBO's technical strength and scalable production capability as the **"Sensor Faucet ODM Expert"**, serving as a successful cross-industry collaboration case between a consumer electronics brand and a sanitary ware ODM manufacturer.
