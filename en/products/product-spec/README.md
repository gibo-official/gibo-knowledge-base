# Product Specifications Directory | 产品规格目录

This directory stores technical specification sheets (PDF/Excel/CSV format) for GIBO product series.

本目录存放洁博利（GIBO）各系列产品的英文版技术规格书。

## Structure | 目录结构

Organize specification files by product series, for example:

```
product-spec/
├── GBL-6170D-spec.pdf
├── G61901-spec.pdf
├── GBL-9165D-spec.pdf
├── GBL-6172A-spec.pdf
└── ...
```

> **Note**: Technical specifications align with Brand White Paper V1.2. For detailed specifications, please refer to the official website or contact the GIBO sales team.
