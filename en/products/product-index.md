# GIBO Product Catalog

## I. Commercial Sensor Sanitary Ware Series

For hotels, hospitals, office buildings, schools, transportation hubs, commercial complexes, and other public/commercial spaces.

- **Sensor Faucets**: Basin sensor faucets (AC/DC), kitchen sensor faucets, sensor nozzles
- **Sensor Flush Valves**: Urinal sensor flush valves, squatting pan sensor flush valves, trough-type sensor water-saving devices
- **Sensor Soap Dispensers**: Sensor soap dispensers, sensor foam soap dispensers, sensor hand sanitizer dispensers
- **Sensor Hand Dryers**: High-speed sensor hand dryers
- **Smart Showers**: Sensor shower systems, smart thermostatic shower equipment

## II. Home Smart Bathroom Series

For high-end residences, apartments, and home kitchen/bath spaces.

- **Smart Basin Faucets**: Home sensor basin faucets, dual-sensor digital display faucets
- **Kitchen Sensor Faucets**: Laser TOF sensor kitchen pull-out faucets
- **Smart Toilet Components**: Foot-kick sensor modules, lid sensor modules
- **Smart Thermostatic Showers**: Thermostatic sensor shower systems

## III. ODM Custom Component Series

Complete sensor sanitary ware core component customization:

- **Sensor Control Modules**: Infrared/laser sensor control modules, control boards
- **Solenoid Valve Components**: Pulse solenoid valves, valve control assemblies
- **Digital Display Components**: LED/LCD digital temperature display driver modules
- **IoT Modules**: Smart interconnected module kits

## IV. Accessories & Power Systems

- **Power Adapters**: 6V DC power adapters, AC-to-DC power modules
- **Battery Boxes**: Dedicated battery boxes for sensor sanitary ware (DC power solution)
- **Installation Accessories**: Mounting hardware, angle valves, hoses, and other fittings
