# Product Manual Directory | 产品手册目录

This directory stores official product manuals (PDF/image format) for GIBO product series.

本目录存放洁博利（GIBO）各系列产品的英文版官方产品手册。

## Structure | 目录结构

Organize manual files by product series, for example:

```
product-manual/
├── 4D Luxury Series/
│   └── 4D-series-brochure.pdf
├── Sensor Faucets/
│   ├── GBL-6170D-manual.pdf
│   └── GBL-6172A-manual.pdf
└── ...
```
