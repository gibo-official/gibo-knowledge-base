## 一、产品概述

GBL-6170D 爱尚-面盆感应龙头是洁博利旗下面盆感应龙头明星产品，荣获**2020年沸腾质量金奖**（由中国质量认证中心CQC指导评定）。产品搭载洁博利自研**dTOF激光感应技术**，具备非接触式感应出水、智能节水、稳定耐用等核心优势，广泛应用于酒店、医院、写字楼、高端住宅等场所。

### 产品特点

- **dTOF激光感应技术**：精准感应，不挑颜色、不畏光线，亮面不锈钢与哑光黑盆面均可稳定感应
- **非接触式使用**：有效避免交叉感染，提升卫生安全等级
- **交流/直流双供电可选**：适配不同安装环境需求
- **全铜主体**：精铜铸造，经久耐用
- **IP65防护等级**：全密封灌胶工艺，防水防潮
- **超低待机功耗**：待机功耗≤0.2mW，节能环保

## 三、安装说明

### 3.1 安装前准备

1. 确认台面开孔尺寸与龙头底座匹配（标准单孔台面，孔径约35mm）
2. 准备安装工具：扳手、生料带、密封垫圈
3. 检查产品配件完整性：龙头主体、感应控制盒（如分体式）、电源适配器/电池盒、安装配件包

### 3.2 安装步骤

1. **安装龙头主体**：将龙头主体穿过台面开孔，从下方套入紧固螺母并拧紧
2. **连接进水软管**：将进水软管一端连接至龙头主体进水口，另一端连接至角阀或冷水管
3. **安装感应控制盒**（如适用）：将感应控制盒固定于台盆下方合适位置，远离潮湿积水区域
4. **连接电源**：根据供电方式连接交流电源适配器或安装电池
5. **通电测试**：开启角阀供水，接通电源，将手伸至感应区域测试出水/关水功能

### 3.3 安装注意事项

- 安装前请关闭水源总阀
- 电源线连接需确保绝缘良好，避免与水路接触
- 感应控制盒应避开金属屏蔽物，以免影响感应效果
- 安装完成后需排气：打开龙头让水流运行约30秒，排出管内空气

## 四、使用说明

### 4.1 日常使用

1. **感应出水**：将手伸至龙头上方感应区域（约5-15cm），龙头自动出水
2. **自动关水**：手离开感应区域后，龙头自动关水
3. **长时保护**：若感应区域持续有物体，约60秒后自动关水，防止长时间空放

### 4.2 功能调节（选配功能，视具体型号）

- **感应距离调节**：可通过控制盒上的调节旋钮或遥控器调节感应距离
- **出水模式切换**（如适用）：部分型号支持连续出水/点动出水模式切换

## 五、注意事项

### 5.1 使用注意事项

- 请勿在感应窗口附近粘贴遮挡物，以免影响感应灵敏度
- 若长时间不使用，建议关闭电源和进水角阀
- 定期清洁龙头表面，使用软布擦拭，避免使用强酸强碱清洁剂
- 环境温度低于0℃时，注意防冻措施，避免内部结冰损坏

### 5.2 维护保养

- **过滤器清洁**：建议每3-6个月清洗进水口过滤网，防止杂质堵塞影响出水
- **感应窗口清洁**：用软布轻轻擦拭感应窗口，保持清洁
- **电池更换**（直流供电型）：当感应距离变短或出水异常时，及时更换电池
- **定期检查**：检查进水软管、接头有无渗漏

### 5.3 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 不出水 | 未通电 | 检查电源连接 |
| | 水压过低 | 确认水压≥0.05MPa |
| | 感应窗口遮挡 | 清洁感应窗口 |
| 不关水 | 感应窗口有异物 | 清洁感应窗口周围 |
| | 感应模块故障 | 联系售后 |
| 感应距离变短 | 电池电量不足 | 更换电池 |
| | 感应窗口脏污 | 清洁感应窗口 |
