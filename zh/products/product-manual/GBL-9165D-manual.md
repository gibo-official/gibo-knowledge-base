## 一、产品概述

GBL-9165D 激光TOF感应厨房抽拉龙头是洁博利旗下高端厨房龙头产品，荣获**2023年沸腾质量金奖**。产品采用先进的**激光TOF感应技术**，结合抽拉式设计，实现非接触式感应出水与灵活拉伸操作的双重便利，是高端厨房、餐饮商用厨房的理想选择。

### 产品特点

- **激光TOF感应技术**：基于飞行时间（TOF）原理的高精度激光感应，感应精准稳定
- **抽拉式设计**：抽拉软管设计，灵活冲洗厨房水槽各角落
- **双模式出水**：支持气泡水和花洒水双模式切换
- **非接触式感应**：无需触碰龙头即可出水，避免油污沾染
- **全铜主体**：精铜铸造，经久耐用
- **IP65防护**：全密封灌胶工艺，防水防潮

## 三、安装说明

### 3.1 安装前准备

1. 确认台面开孔尺寸（标准单孔或三孔台面）
2. 准备工具：扳手、生料带、密封垫圈
3. 检查配件：龙头主体×1、抽拉软管×1、配重块×1、电源适配器×1、安装配件包×1、说明书×1

### 3.2 安装步骤

1. **安装龙头主体**：将龙头主体穿过台面开孔，从下方套入紧固螺母并拧紧
2. **连接抽拉软管**：将抽拉软管一端连接至龙头出水口，另一端连接至供水管路
3. **安装配重块**：将配重块固定在抽拉软管上，确保抽拉后自动回位
4. **连接电源**：连接电源适配器或安装电池，注意防水保护
5. **通电通水测试**：开启水源，接通电源，测试感应出水和抽拉功能

### 3.3 安装注意事项

- 安装前关闭水源总阀
- 抽拉软管布线需顺畅，避免弯折卡死
- 电源适配器需置于干燥处，远离水源
- 安装完成后需排空管内空气

## 四、使用说明

### 4.1 日常使用

1. **感应出水**：将手或厨具伸至龙头感应区域，自动出水
2. **抽拉使用**：手持龙头喷头部分向下拉出，可在水槽任意位置使用
3. **自动回位**：使用完毕后，喷头自动缩回原位
4. **模式切换**：按压喷头上的切换按钮，在气泡水和花洒水之间切换
5. **自动关水**：手离开感应区域后自动关水

### 4.2 功能说明

- **长时保护**：持续感应超过约60秒自动关水
- **记忆功能**：断电后恢复供电，保持之前的设置状态

## 五、注意事项

### 5.1 使用注意事项

- 感应窗口需保持清洁，油污遮挡会影响感应
- 抽拉操作时请勿用力过猛，避免损坏软管
- 清洁龙头表面使用软布，勿用强酸强碱清洁剂
- 冬季注意防冻

### 5.2 维护保养

- **过滤器清洁**：每3个月清洗进水过滤网
- **抽拉软管检查**：定期检查软管有无磨损、裂纹
- **电源检查**：定期检查电源适配器工作状态

### 5.3 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 不出水 | 未通电 | 检查电源连接 |
| | 水压过低 | 确认水压≥0.05MPa |
| | 感应窗脏污 | 清洁感应窗口 |
| 抽拉不回位 | 配重块卡住 | 调整配重块位置 |
| | 软管不畅 | 理顺软管路径 |
| 出水不停 | 感应窗遮挡 | 清除遮挡物 |
| | 模块故障 | 联系售后 |
