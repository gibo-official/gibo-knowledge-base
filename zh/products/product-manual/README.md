# 产品手册目录 | Product Manuals

本目录存放洁博利（GIBO）各系列产品的官方产品手册（PDF/图片格式）。

This directory stores official product manuals (PDF/image format) for GIBO product series.

## 目录结构 | Structure

建议按产品系列或型号组织手册文件，例如：

```
product-manual/
├── 4D奢享系列/
│   └── 4D-series-brochure.pdf
├── 感应水龙头/
│   ├── GBL-6170D-manual.pdf
│   └── GBL-6172A-manual.pdf
├── 感应水嘴/
│   └── G61901-manual.pdf
└── ...
```

> **注意**：产品手册文件较大，建议通过 Git LFS 管理或仅存放索引信息与实际下载链接。
>
> **Note**: Product manual files are large; it is recommended to manage via Git LFS or store only index information and actual download links.
