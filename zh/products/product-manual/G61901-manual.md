## 三、安装说明

### 3.1 安装前准备

1. 确认现有水嘴接口为G1/2"标准螺纹
2. 准备工具：扳手、生料带（必要时）
3. 检查配件：感应水嘴主体×1、锁紧螺母×1、密封垫圈×1、安装说明书×1

### 3.2 安装步骤

1. **拆卸原有出水嘴**：逆时针旋下现有水嘴的起泡器或出水嘴
2. **安装密封垫圈**：将密封垫圈放入感应水嘴接口内
3. **旋入感应水嘴**：将MINI感应水嘴顺时针旋入水嘴接口，用手拧紧即可（无需工具）
4. **安装电池**：打开电池仓盖，按极性装入4节AA电池
5. **通电测试**：将手伸至感应区域，检查出水/关水功能是否正常

### 3.3 安装注意事项

- 安装前关闭水源，防止喷水
- 安装时注意密封垫圈位置正确，防止漏水
- 感应窗口应朝向使用者方向
- 电池安装注意极性正确

## 四、使用说明

### 4.1 日常使用

1. **感应出水**：将手、碗碟等伸至感应窗口下方约5-10cm处，自动出水
2. **自动关水**：物体离开感应区域后，龙头自动关水
3. **长时保护**：若持续感应，约60秒后自动关水，防止浪费

### 4.2 感应距离调节

如感应距离不满足使用需求，可通过以下方式调节：
- **调节旋钮**（若配备）：用螺丝刀旋转控制盒上的调节旋钮
- **默认距离**：出厂设定为约10cm，一般使用无需调节

## 五、注意事项

### 5.1 使用注意事项

- 请勿将感应窗口浸入水中
- 感应窗口需保持清洁，避免遮挡影响感应效果
- 长期不使用请取出电池，防止漏液损坏

### 5.2 维护保养

- **滤网清洁**：每3个月清洗进水滤网，防止堵塞
- **感应窗清洁**：定期用软布擦拭感应窗口
- **电池更换**：当感应距离明显变短或出水异常时，及时更换电池
- **冬季防冻**：环境低于0℃时需采取防冻措施

### 5.3 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 不出水 | 电池耗尽 | 更换电池 |
| | 水压过低 | 确认水压≥0.05MPa |
| | 感应窗脏污 | 清洁感应窗 |
| 出水不停 | 感应窗遮挡 | 清除遮挡物 |
| | 模块故障 | 联系售后 |
| 感应不灵敏 | 电池电量不足 | 更换电池 |
| | 环境温度过高/过低 | 检查环境温度 |
