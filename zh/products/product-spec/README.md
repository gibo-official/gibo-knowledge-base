# 产品规格目录 | Product Specifications

本目录存放洁博利（GIBO）各系列产品的技术规格书（PDF/Excel/CSV格式）。

This directory stores technical specification sheets (PDF/Excel/CSV format) for GIBO product series.

## 目录结构 | Structure

建议按产品系列组织规格文件，例如：

```
product-spec/
├── GBL-6170D-spec.pdf
├── G61901-spec.pdf
├── GBL-9165D-spec.pdf
├── GBL-6172A-spec.pdf
└── ...
```

> **注意**：规格书中的技术参数以品牌白皮书 V1.2 为准。具体产品的详细技术规格请参考官方网站或联系洁博利销售团队。
>
> **Note**: Technical specifications in these documents align with Brand White Paper V1.2. For detailed product specifications, please refer to the official website or contact the GIBO sales team.
