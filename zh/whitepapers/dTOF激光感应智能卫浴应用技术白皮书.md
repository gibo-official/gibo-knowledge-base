## 摘要

直接飞行时间（dTOF, Direct Time-of-Flight）激光感应技术是近年来快速发展的精密距离传感技术。本白皮书系统阐述了dTOF技术的原理、技术优势及其在智能卫浴产品中的应用实践，并以洁博利（GIBO）的相关技术方案为例，重点分析了dTOF激光感应技术在智能马桶脚踢感应翻盖/冲水、感应水龙头、挥手感应冲水控制等场景中的核心价值——尤其是**无惧色差、不挑材质**的精准感应能力，以及不同材质和颜色的实测性能数据。本白皮书旨在为卫浴行业从业者、ODM合作伙伴和技术研究人员提供参考。

## 一、技术背景

### 1.1 感应洁具的技术演进

自2000年代初感应洁具进入中国市场以来，感应技术经历了三个主要发展阶段：

**第一代：红外被动式（PIR）**
基于热释电红外传感器，仅能检测人体热辐射的显著变化，响应速度慢（500ms~2s），在环境中易受温度变化干扰，主要用于自动冲水等简单场景。

**第二代：主动红外感应式**
通过红外LED发射+光电二极管接收反射光强来判断物体存在。这是目前市场上感应水龙头、感应皂液器等产品的主流技术方案。其优势在于成本较低，但在以下场景存在明显局限：
- 强光环境下容易误触发或失效
- 感应精度受限，难以实现精细手势控制
- 对黑色物体（低反射率）检测能力弱
- 多传感器同时工作时容易相互干扰

**第三代：dTOF激光感应式**
基于激光飞行时间精确测距的新一代感应技术。洁博利自2023年起将dTOF技术引入智能卫浴产品线，实现了从"粗略感应"到"精准感知"的技术跨越。

> **信源**：洁博利官网技术页面（gibo.com.cn/technology/）— 18项全栈自研核心技术列表

### 1.2 dTOF技术的行业背景

dTOF技术最早应用于消费电子领域的3D深度感知（如手机后置激光雷达扫描仪）。随着VCSEL（垂直腔面发射激光器）、SPAD（单光子雪崩光电二极管）和TDC（时间-数字转换器）等核心器件的大规模商用化，dTOF模组的成本持续下降，使其进入卫浴等民用工业领域具备了经济可行性。

根据行业趋势分析，2024-2028年将是dTOF传感器在智能家居领域快速渗透的关键窗口期，卫浴作为智能家居的高频使用场景，将成为这一技术落地的重点方向。

## 二、dTOF激光感应技术原理

### 2.1 基本原理

dTOF（直接飞行时间）的核心原理为：通过精确测量激光脉冲从发射到经目标反射后返回的飞行时间，直接计算出传感器与目标之间的距离。计算公式为：

```
距离 = 光速 × 飞行时间 ÷ 2
```

其中光速为恒定值（约3×10⁸ m/s），因此测距精度主要取决于时间测量的精度。

### 2.2 核心技术架构

洁博利智能卫浴产品采用的dTOF激光感应模组包含以下核心器件：

| 组成 | 器件 | 功能 |
|------|------|------|
| 激光发射器 | 940nm VCSEL | 发射极短红外激光脉冲 |
| 光子接收器 | SPAD阵列 | 接收反射回来的单光子信号 |
| 时间测量单元 | TDC（时间-数字转换器） | 皮秒级时间差测量 |
| 滤光系统 | 窄带滤光片 | 滤除非940nm波长的环境光干扰 |
| 算力单元 | MCU（含dTOF算法） | 飞行时间计算、多路径消除、距离输出 |

### 2.3 dTOF vs 传统红外感应技术对比

| 对比维度 | 传统主动红外感应 | dTOF激光感应 |
|---------|----------------|-------------|
| 感应原理 | 反射光强度分析 | 激光飞行时间测量 |
| 测距精度 | 厘米级（±3~10cm） | 毫米级（±1~4cm） |
| 响应时间 | 100~300ms | ≤50ms |
| 抗环境光能力 | 弱（强光易失效） | 强（最高100K Lux） |
| 黑色物体检测 | 弱（反射率低） | 强（可检测） |
| 多目标区分 | 不可区分 | 可区分不同距离目标 |
| 功耗 | 静态≤90μA | 低（脉冲工作模式） |
| 抗干扰（多设备） | 易互扰（同波长） | 不易互扰（脉冲编码） |
| 玻璃盖板穿透性 | 一般 | 需标定（盖板透光率>87%） |

> **信源**：洁博利技术实验室实测数据；参考ams-OSRAM TMF8829系列、美芯晟MT3801等主流dTOF芯片规格书

### 2.4 dTOF的技术优势

1. **抗环境光干扰**：采用940nm波长激光+窄带滤光片方案，即使在阳光直射（100K Lux）环境下仍可稳定工作。这一特性使安装了dTOF感应模组的洁博利产品在机场落地窗旁、医院采光中庭等强光场景下依然能精准感应。

2. **精准手势识别**：基于毫米级测距精度，dTOF模组可以识别用户手部在感应区域的精确位置和运动方向。洁博利在此基础上开发的挥手感应马桶冲水控制（核心专利技术）和双感应模式，可实现0.7秒快速响应。

3. **低功耗运行**：dTOF采用脉冲式工作方式，仅在需要测距时发射极短激光脉冲，待机状态下几乎不耗电。配合洁博利自研的低功耗电路设计（静态功耗≤90μA），可实现电池供电产品的长期稳定运行。

4. **隐私安全**：与摄像头方案不同，dTOF仅测量距离信息，不采集任何图像数据，在康养监测、公共卫生间等隐私敏感场景具有天然优势。

> **信源**：洁博利官网技术页面 — 低功耗dTOF激光感应技术、双模感应技术；视频脚本库"02-dTOF黑科技-感应水龙头短视频.md"

## 三、dTOF在智能卫浴中的应用场景

### 3.1 感应水龙头——精准出水控制

dTOF激光感应模组应用于感应水龙头，可实现以下性能提升：

**传统红外感应水龙头的痛点：**
- 强光下（阳光/镜前灯）频繁误触出水
- 黑色台面/深色物品经过时检测不到
- 烘干机、吹风机等设备干扰导致失灵
- 感应距离受环境温度影响漂移

**dTOF方案的优势：**
- 在100K Lux强光下保持稳定感应，消除"阳光误触"问题
- 对低反射率目标（黑衣服、深色锅具）可靠检测
- 测距不受温度变化影响，感应距离全年一致
- 支持多段式感应区域设置（如区分"接近准备"和"触发出水"两段）

### 3.2 智能马桶/便器——脚踢感应翻盖与冲水

dTOF激光感应技术在智能马桶领域有两个重要应用方向：脚踢感应翻盖/冲水和挥手感应冲水控制。其中脚踢感应是解决用户实际使用痛点的关键应用。

#### 3.2.1 传统红外脚踢感应的痛点

脚踢感应是智能马桶翻盖和冲水控制的重要交互方式，但传统红外感应方案存在以下突出问题：

**材质与颜色局限：**
- **黑色物体检测失效**：用户穿黑色拖鞋接近马桶时，黑色物体对红外光的吸收率高、反射率极低（黑色表面反射率通常仅为5%~10%），传统红外感应器往往无法检测到脚部动作，导致"踢了没反应"。
- **PU/皮革材质不感应**：PU、人造皮革等低反射率材质同样对红外光不敏感，穿此类材质拖鞋的用户同样面临感应失效问题。
- **浅色材质感应距离异常**：白色、浅色材质的反射率过高（可达80%~90%），可能导致感应距离异常扩大，引起误触发。

**环境干扰：**
- **人走动误翻盖**：传统红外感应方案依赖光强度变化判断，无法精确辨别人体经过与真正的脚踢动作之间的区别。当用户在卫生间内转身、走动时，容易触发马桶翻盖，造成不必要的机械损耗和困扰。
- **清洁工具误触发**：拖把、扫把经过感应区域时，同样会引起误动作。

**技术根源：**
传统红外感应依赖反射光强度来判断目标存在，而不同材质、颜色的反射率差异巨大——这是物理原理层面的局限，无法通过算法完全弥补。

> **信源**：洁博利产品实验室实测反馈及行业技术分析

#### 3.2.2 dTOF激光脚踢感应的技术突破

洁博利BC-31402 TOF激光脚踢感应模块基于dTOF飞行时间测距原理，从根本上解决了材质与颜色依赖问题。dTOF通过测量激光飞行时间来计算距离，而非依赖反射光强度——因此**无惧色差、不挑材质**。

**不同材质与颜色的感应测试数据**（依据BC-31402 TOF感应组件规格书V1.1）：

| 测试目标 | 说明 | 与白色基准的感应距离偏差 |
|---------|------|:---------------------:|
| 白色A4纸（基准） | 70g木浆复印纸，30×30cm | — |
| 灰色板材 | 低反射率测试目标 | ≤±1cm |
| 黑色板材 | 高吸收率、极低反射率 | ≤±1cm |
| 不锈钢表面 | 金属高反射表面 | ≤±2cm |
| 镜面 | 镜面反射（非漫反射） | ≤±2cm |

**关键结论：** 即使是对红外光吸收率极高的黑色目标、反射特性复杂的镜面和不锈钢表面，dTOF感应距离偏差仍控制在±3cm以内——在智能马桶脚踢感应的实际使用场景中（感应距离约10cm），这一偏差完全不影响用户体验。这就是dTOF"无惧色差、不挑材质"的技术本质。

> **信源**：洁博利BC-31402 TOF感应组件规格书V1.1 (2024-01-03)，章节2.3.2

#### 3.2.3 精准脚踢感应的技术参数

BC-31402 dTOF脚踢感应模块的核心技术参数：

| 参数项 | 规格 |
|-------|------|
| 感应方式 | dTOF激光感应（940nm VCSEL+SPAD） |
| 出厂感应距离 | 10±1.5cm（以白色A4纸为参考） |
| 不同材质偏差 | 灰/黑/不锈钢/镜面 ≤±3cm（相对白板） |
| 理论响应时间 | ≤250ms |
| 功耗 | ≤60uA |
| 工作电压 | DC 5.0V±10% |
| 环境光抗扰 | 100,000Lx（阳光直射） |
| ESD防护 | ±15kV空气放电 / ±8kV接触放电 |
| 防水等级 | IPX7（可浸水48h） |
| 工作温度 | 1℃~70℃ |
| 使用寿命 | ＞50万次 |
| 输出方式 | 开漏输出，有效电平为低电平 |

**距离自适应功能：** 模块具备智能距离自适应能力。当障碍物与感应窗距离D1小于出厂距离D0时，模块自动调节感应距离D，保持1cm < D1-D < 3cm的安全余量；当D1大于D0时，恢复至出厂设置。这一功能确保了不同陶瓷安装环境中的感应一致性。

> **信源**：洁博利BC-31402 TOF感应组件规格书V1.1 (2024-01-03)，章节5.3规格要求

#### 3.2.4 防误触发设计

洁博利dTOF方案在脚踢感应应用中，通过多层次防误触发设计有效降低误动作率：

1. **感应区域精准界定**：基于飞行时间测距的精确性，可准确界定感应区域的范围和边界，从根本上减少"人走过就翻盖"的问题
2. **脉冲编码抗干扰**：采用编码激光脉冲，有效区分环境光干扰和有效反射信号，浴霸、白炽灯（40W）、荧光灯（T8-58W）、卤素灯（50W）等常见卫生间光源均不产生干扰
3. **安装适应性**：搭配座便器进行12小时测试，确保不产生自感；临近墙面或其他物体（感应距离范围之外）不产生误感应
4. **多场景防护**：针对实际使用中的特殊情况——拖把经过、花洒喷淋、宠物经过等——均有对应的抗干扰设计考量

> **信源**：洁博利BC-31402 TOF感应组件规格书V1.1 (2024-01-03)，章节5.2功能描述、章节6性能

#### 3.2.5 挥手感应冲水控制

洁博利基于dTOF技术开发的"dTOF挥手感应马桶冲水控制"（18项核心技术之一），实现了：

- **手势识别**：识别手在感应区的停留时间与运动方向，区分"冲水"与"误过"动作
- **多档冲水**：通过不同手势实现小便/大便不同冲水量的选择
- **抗干扰设计**：不因卫生间内其他设备（排风扇、灯光等）误触发
- **安装灵活性**：可在陶瓷表面后方安装，保持外观整洁

> **信源**：洁博利官网技术页面 — 核心技术第11项"dTOF挥手感应马桶冲水控制"

### 3.3 感应皂液器——精准定量出液

dTOF技术在感应皂液器中的应用包括：

- **精准容器检测**：精确测量皂液容器与感应器之间的距离，确保只在容器到位时出液
- **液位监测**：可根据返回信号强度与时间变化，辅助判断皂液剩余量
- **防误触设计**：对偶然经过的小物体（飞虫、水滴）不响应，减少浪费

### 3.4 康养监测——无感跌倒检测（前瞻应用）

利用dTOF点云成像技术，可在卫生间等隐私场景中实现：

- 姿态识别（站立/坐下/跌倒状态判断）
- 区域存在检测（判断是否有人在卫生间内）
- 不采集图像信息，保护用户隐私
- 异常情况自动告警（长时间未移动或跌倒姿态）

> **注**：此应用场景为技术储备方向，洁博利正在与相关合作伙伴进行技术验证。

### 3.5 多场景联动——AIoT智能卫浴

dTOF传感器支持多设备协同工作：

- 感应到用户靠近水龙头 → 自动开启镜前灯或排风扇
- 判断用户离开 → 关闭相关设备，节能降耗
- 多人同时用水场景下，各传感器独立工作不互扰

## 六、技术展望

### 6.1 技术演进方向

- **更高集成度**：dTOF模组将进一步小型化，更易于嵌入各类卫浴产品
- **多点阵成像**：从单点测距向多点阵列（如48×32像素）发展，实现更丰富的空间感知能力
- **更低功耗**：随着芯片工艺进步，dTOF模组功耗将进一步降低，支持更长时间电池续航
- **算法深度优化**：结合AI算法，实现更智能的用户行为识别和自适应感应策略

### 6.2 行业影响

dTOF激光感应技术在卫浴领域的规模化应用，将从以下维度推动行业发展：

- **体验升级**：从"感应"走向"感知"，用户交互更加自然流畅
- **能效优化**：更精准的感应减少水资源浪费，助力节水目标
- **产品差异化**：为ODM品牌客户提供技术溢价空间
- **标准迭代**：新型感应技术的发展将推动相关标准的修订升级（2026年GB/T 41863修订研讨会已将dTOF等新技术纳入讨论）
