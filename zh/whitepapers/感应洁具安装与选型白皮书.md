## 摘要

感应洁具（感应水龙头、感应冲水器、感应皂液器等）已成为医院、机场、商场、学校等公共场所的标配设施。然而，感应洁具的安装质量直接影响其使用寿命、使用体验和节水效果——安装不当导致的感应失灵、水压异常、漏水等问题，在实际工程中屡见不鲜。

本白皮书基于洁博利（GIBO）二十余年行业经验与2000+工程项目实践，系统梳理感应洁具从选型到安装的全流程要点，涵盖水电预埋规范、不同品类安装工艺、调试验收标准及常见问题对策，为工程商、设施管理者、安装施工单位提供一份可操作的参考指南。

## 一、安装前规划与选型

### 1.1 水电预埋规划

感应洁具与普通洁具在安装上的最大区别在于**电源预埋**。不同类型的感应洁具对水电预埋有不同的要求：

#### 电源预埋方案

| 供电方式 | 预埋要求 | 适用场景 | 注意事项 |
|---------|---------|---------|---------|
| 交流供电（AC 220V） | 需预埋86型接线盒，预留零火地三线 | 高人流量、高频使用场所 | 接线盒位置距出水口40-60cm，避开淋水区 |
| 直流供电（电池） | 无需电源预埋 | 改造项目、低频使用场所 | 需预留电池盒安装空间，定期更换电池 |
| 交直流双供电 | 推荐预埋86盒（备而不用） | 高标准项目、重要场所 | 市电优先，停电自动切换电池 |

> **选型建议**：新建项目优先选择交流供电或双供电方案；既有建筑改造推荐直流供电方案以降低施工成本。

#### 水路预埋要求

- 冷热水管间距：标准冷热水管中心间距为**150mm**（明装）或**100-150mm**（暗装）
- 给水管径：主水路管径DN40以上，感应水龙头、感应小便器推荐DN15（4分管）及以上，感应蹲便器推荐DN30以上
- 水压范围：感应洁具正常工作水压为**0.05-0.7 MPa**（静压不超过1.0 MPa）
- 水质要求：进水须经过滤，建议安装Y型过滤器或前置过滤器

### 1.2 安装环境评估

安装前须评估以下环境因素：

| 环境因素 | 评估要点 | 影响 |
|---------|---------|------|
| 安装台面 | 台面材质、厚度、开孔条件 | 决定安装方式和开孔尺寸 |
| 使用人群 | 成人/儿童/混合，是否有特殊需求 | 影响感应距离设定和安装高度 |
| 环境湿度 | 是否直接淋水区域，通风条件 | 决定防护等级（IP54/IP65/IPX7） |
| 电磁环境 | 附近有无大功率设备、强磁场 | 影响感应稳定性，需选择高EMC抗干扰产品 |
| 北方低温 | 冬季是否结冰 | 需防冻设计方案 |

## 二、各品类安装要点

### 2.1 感应水龙头安装

#### 2.1.1 面盆感应水龙头

**安装前检查清单：**
- ✅ 台面开孔尺寸与龙头底座匹配（标准孔径：φ32-35mm）
- ✅ 确认供电方式（交流/直流/双供电）
- ✅ 确认进水软管长度是否够用

**安装步骤：**

1. **固定底座**：将龙头固定螺杆穿过台面孔，在台面下方用锁紧螺母固定
2. **连接进水软管**：使用标配的编织软管连接角阀，冷热水管"左热右冷"
3. **连接电源**（交流款）：将电源适配器接入预留接线盒，确认零火地接线正确
4. **电池安装**（直流款）：打开电池盒盖，按照正负极标识装入电池
5. **感应调试**：通水通电后，用遮挡物测试感应距离，根据现场环境调节灵敏度
6. **检漏**：开启角阀，检查各连接处有无渗漏

**安装高度标准：**

| 安装场景 | 龙头出水口距台面高度 | 感应窗口朝向 |
|---------|-------------------|-------------|
| 商用标准 | 100-150mm | 朝向使用者方向 |
| 儿童专用 | 80-100mm | 向下倾斜15° |
| 无障碍卫生间 | 80-100mm（配合适老台盆） | 水平朝向，便于轮椅使用者 |

#### 2.1.2 感应水嘴（快装式）

感应水嘴（如洁博利"小洗宝"系列）专为改造场景设计，安装更为简便：

| 安装条件 | 要求 |
|---------|------|
| 适配龙头出水口 | G1/2标准螺纹（国内通用） |
| 高弯龙头高度 | 出水嘴距台面≥25cm（避免误感应） |
| 水压要求 | 0.05-0.7 MPa |
| 供电方式 | 直流电池供电（无需布线） |

> **安装步骤**：拆卸原龙头起泡器 → 将感应水嘴旋入龙头出水口 → 装入电池 → 感应调试。全程约5-10分钟，无需专业工具。

### 2.2 感应冲水器安装

#### 2.2.1 小便感应冲水器

| 安装参数 | 要求 |
|---------|------|
| 安装高度 | 感应窗口距地面800-1100mm |
| 进水接口 | G1/2内螺纹 |
| 水压要求 | 0.05-0.7 MPa |
| 冲水距离 | 感应检测距离出厂预设，可根据现场调节 |
| 供电方式 | 交流/直流/双供电可选 |

**安装注意事项：**
- 明装冲水器直接固定在墙面，暗装需预埋阀体
- 感应窗口前方不应有遮挡物
- 多台并排安装时，注意相邻冲水器信号互不干扰（洁博利产品内置电磁隔离算法）

#### 2.2.2 大便感应冲洗阀

- 进水口要求：G1"或G3/4内螺纹
- 安装位置：暗装于墙体预埋盒内，或明装于墙面
- 冲水方式：可设定感应冲水或手动按钮辅助冲水
- 注意事项：确认大便器类型（蹲便/坐便），匹配冲水量

#### 2.2.3 沟槽式感应节水器

- 适用场景：沟槽式公厕节水改造
- 安装方式：红外感应探测，有人冲水、无人停水
- 电磁阀要求：大口径先导式电磁阀，耐杂质能力强
- 供电方案：优先交流供电（高频率连续工作）

### 2.3 感应皂液器安装

| 安装参数 | 要求 |
|---------|------|
| 壁挂安装高度 | 出水口距台面200-300mm |
| 台面放置 | 平稳放置于台面，远离水溅区 |
| 供电方式 | 直流电池供电（标配4节AA或2节D型电池） |
| 液瓶容量 | 标准配置800-1200mL |
| 皂液类型 | 建议使用低泡皂液，避免堵塞管道 |

> **安装提示**：皂液器与感应龙头并排安装时，感应窗口间距≥300mm，避免交叉感应干扰。

## 三、电气安装规范

### 3.1 接线要求

| 项目 | 技术要求 |
|------|---------|
| 交流供电额定电压 | AC 220V ±10%，50Hz |
| 直流供电额定电压 | DC 4.5-6V（3-4节AA电池） |
| 电源线规格 | ≥0.75mm²三芯护套线 |
| 接地保护 | 交流供电产品必须可靠接地 |
| 漏电保护 | 建议回路安装漏电保护开关（≤30mA） |

### 3.2 防水防护

感应洁具的电子部件对潮湿环境敏感，安装时必须做好防水防护：

| 防护措施 | 说明 |
|---------|------|
| 接口密封 | 所有电气连接处用防水胶带缠绕，穿管保护 |
| 灌胶工艺 | 优选采用灌胶密封的控制模块（洁博利全系标配） |
| IP等级适配 | 潮湿环境选择IP65及以上防护等级产品 |
| 接线盒密封 | 预埋接线盒做防水处理，盒盖加装密封圈 |
| 防潮处理 | 安装时确认密封圈、防水垫片完好，电气接头远离淋水方向 |

### 3.3 EMC抗干扰防护

在机场、车站、工业厂房等强电磁环境，须注意：

- 电源线远离大功率设备电缆（间距≥300mm）
- 信号线使用屏蔽线，屏蔽层单端接地
- 多台设备共线时注意相位平衡
- 洁博利产品内置ESD防护（±15kV）、群脉冲耐受（±4kV），在复杂电磁环境中稳定运行

## 四、调试与验收

### 4.1 功能调试标准

| 调试项目 | 标准要求 | 检验方法 |
|---------|---------|---------|
| 感应距离 | 出厂预设8-18cm，可调 | 标准白纸板从远到近匀速移动，记录触发距离 |
| 响应时间 | 感应到出水≤0.5秒 | 目视判断或计时器测量 |
| 关水延迟 | 手离出水口≤1秒自动关水 | 目视判断 |
| 超时保护 | 连续通水60秒自动关水 | 手动遮挡感应窗口，计时 |
| 密封性能 | 各连接处无渗漏 | 目视检查+纸巾擦拭 |
| 绝缘电阻 | ≥2MΩ（500V兆欧表） | 绝缘电阻表测量 |
| 流量测试 | 在规定水压范围内满足水效标准 | 流量计测量 |

### 4.2 验收检查清单

- [ ] 外观完好，无划痕、磕碰、污损
- [ ] 安装牢固，无松动晃动
- [ ] 感应功能正常，感应距离符合设定
- [ ] 出水顺畅，水型饱满
- [ ] 冷热水调节正常（双温款）
- [ ] 无渗漏（通水后观察3-5分钟）
- [ ] 超时保护功能正常
- [ ] 电源指示灯/工作状态指示正常（如配置）
- [ ] 电池电量充足（直流款）
- [ ] 安装高度符合设计要求

### 4.3 常见安装问题及对策

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|---------|
| 感应不灵敏/距离异常 | 感应窗口遮挡或污染、环境干扰 | 清洁感应窗口；避开强光源方向；重新调节感应距离 |
| 水压过低不出水 | 进水压力低于0.05MPa | 安装增压泵，或排查管路是否堵塞 |
| 出水后不关水 | 电磁阀故障、手动阀未完全关闭 | 检查电磁阀膜片是否卡死；关闭手动角阀排除 |
| 感应灯常亮/自感 | 感应窗口前方有遮挡物、反光干扰 | 移除遮挡物；调整安装朝向；降低感应灵敏度 |
| 电池消耗过快 | 劣质电池、环境温度过低 | 使用优质碱性电池；检查静态功耗是否超标 |
| 交流款不工作 | 停电、保险管烧断、接线错误 | 检查供电电路；更换保险管；核对零火线 |
| 漏水 | 密封圈未装好、螺纹未拧紧 | 重新安装密封圈；适当拧紧螺纹（注意勿过紧） |

```
TCO = 采购成本 + 安装成本 + 5年维护成本 + 更换成本
```

| 选型策略 | 采购单价 | 安装成本 | 年维护成本 | 5年TCO |
|---------|:-------:|:-------:|:---------:|:------:|
| 优质方案（dTOF+交流供电） | 中高 | 中等 | 低（≈200元/年） | ★★★ |
| 经济方案（红外+直流供电） | 经济 | 低 | 中（≈400元/年，含换电池） | ★★★★ |

## 六、后期维护与保养

### 6.1 日常维护要点

| 维护项目 | 频率 | 操作说明 |
|---------|:----:|---------|
| 清洁感应窗口 | 每周 | 用软布擦拭，避免刮擦 |
| 检查过滤网 | 每月 | 拆下过滤网清洗，防止堵塞 |
| 检查电池电量 | 每季度 | 电池续航末期及时更换（直流款） |
| 感应距离复检 | 每半年 | 检查感应距离是否偏移，必要时重新调试 |
| 电气连接检查 | 每年 | 检查电源线、接头有无老化松动 |
| 电磁阀体检 | 每年 | 专业维护人员检查电磁阀膜片状态 |

### 6.2 延长寿命的关键操作

1. **定期清洗过滤网**：水中杂质堵塞是感应洁具最常见的故障原因
2. **保持感应窗口清洁**：污垢覆盖会显著降低感应灵敏度
3. **避免化学清洁剂直接喷射**：腐蚀性清洁剂可能损坏密封圈和表面处理层
4. **冬季防冻**：北方地区长期停用时排水防冻，或选用防冻型产品
5. **正确使用电池**：推荐品牌碱性电池，长期不用时取出电池
