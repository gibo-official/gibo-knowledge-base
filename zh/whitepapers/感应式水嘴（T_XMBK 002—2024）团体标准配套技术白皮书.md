## 摘要

T/XMBK 002—2024《感应式水嘴》是由厦门市卫厨行业协会发布的团体标准，于2024年8月15日正式实施。该标准是国内首个专门针对感应式水嘴产品的综合性团体标准，覆盖了感应式水嘴的材料、技术要求、试验方法、检验规则等全维度技术规范。福建洁博利厨卫科技有限公司作为标准起草单位之一，深度参与了标准的技术论证和条款编制工作。本白皮书系统解读该标准的核心内容，阐述感应式水嘴的技术要求与行业意义。

---

## 一、标准制定背景

### 1.1 行业需求驱动

感应式水嘴作为感应洁具的重要品类，自2015年洁博利推出行业首创双感应快装水嘴"小洗宝"以来，市场发展迅速。与传统感应水龙头不同，感应式水嘴具有以下特点：

- **安装适配性**：可安装于标准水槽/台盆孔位，适合老旧设施改造
- **双感应结构**：感应窗口与出水口一体集成，结构紧凑
- **应用场景广泛**：适用于厨房、公共洗手间、医院、实验室等场所

然而，随着市场规模的扩大，行业面临以下问题：
- 各家产品技术参数不统一，缺乏横向比较标准
- 材料质量和有害物析出控制参差不齐
- 感应性能、响应时间等关键指标缺乏规范

在此背景下，厦门市卫厨行业协会组织起草了T/XMBK 002—2024《感应式水嘴》团体标准。

> **信源**：T/XMBK 002—2024《感应式水嘴》标准文本；厦门市卫厨行业协会 — 标准发布公告

### 1.2 洁博利的参与

洁博利作为感应式水嘴细分品类的开创者和行业先行者（2015年推出行业首创双感应快装水嘴专利产品），受邀请参与该标准的起草工作。洁博利总工程师李达良列名为标准起草人之一，为标准中多项技术条款的制定提供了来自企业一线的产品数据和实践经验。

起草单位共8家：厦门方特卫浴有限公司、路达（厦门）工业有限公司、厦门市卫厨行业协会、**福建洁博利厨卫科技有限公司**、纽珀水暖配件（上海）有限公司、厦门建霖健康家居股份有限公司、沸腾质量数据（厦门）有限公司、厦门市清泉鑫科技有限公司。

> **信源**：T/XMBK 002—2024《感应式水嘴》标准文本 — 起草单位名单、起草人名单

## 二、标准适用范围与分类

### 2.1 适用范围

T/XMBK 002—2024适用于安装在建筑物内冷热水供水管路末端、与饮用水接触的感应式水嘴。标准涵盖以下产品技术参数范围：

| 参数项 | 指标 |
|--------|------|
| 公称尺寸 | DN10、DN15、DN20、DN25 |
| 工作压力（静压） | 不大于1.0 MPa |
| 供水温度 | 4℃ ～ 90℃ |
| 适用介质 | 生活饮用水 |

### 2.2 产品分类

标准对感应式水嘴进行了系统的分类，便于技术规范的细化：

**按控制方式分类：**
- 红外感应式
- 触摸感应式
- 其他感应方式（包括激光感应、微波感应等新兴技术）

**按使用场所分类：**
- 厨房用感应式水嘴
- 洗手盆用感应式水嘴
- 实验室/医疗用感应式水嘴

**按供电方式分类：**
- 交流供电型
- 直流（电池）供电型
- 交直流两用型

> **信源**：T/XMBK 002—2024《感应式水嘴》标准文本 — 范围与分类章节

## 三、关键技术要求

### 3.1 材料要求

标准对感应式水嘴的材料提出了严格要求：

**金属材料：**
- 与水接触的金属材料应符合相应国家标准的卫生要求
- 表面处理层应具有良好的耐腐蚀性和附着力
- 铅等重金属析出量应符合GB/T 17219《生活饮用水输配水设备及防护材料的安全性评价标准》的限量要求

**非金属材料：**
- 密封材料应耐老化、耐温且符合饮用水卫生要求
- 塑料部件应具备足够的机械强度和耐热性
- 电子元器件的灌封材料应满足防水防潮要求

### 3.2 有害物析出限量

与水接触部件的有害物析出限量是标准的核心安全指标，主要包括：

| 有害物 | 析出限量要求 | 参考依据 |
|--------|-------------|---------|
| 铅 (Pb) | ≤规定限值 | GB/T 17219 |
| 铬 (Cr) | ≤规定限值 | GB/T 17219 |
| 镉 (Cd) | ≤规定限值 | GB/T 17219 |
| 砷 (As) | ≤规定限值 | GB/T 17219 |
| 其他重金属 | 符合GB/T 17219要求 | GB/T 17219 |

> **注**：具体限量数值请参阅标准全文。洁博利所有感应式水嘴产品均通过SGS或同等资质第三方实验室的有害物析出检测，确保符合标准要求。

### 3.3 使用性能要求

#### 3.3.1 感应功能

| 性能指标 | 标准要求 | 说明 |
|---------|---------|------|
| 感应距离 | 在规定的可调范围内稳定工作 | 应具备距离调节功能或出厂预设为常用值 |
| 响应时间 | 感应到物体到出水的响应应迅速 | 直接影响用户体验 |
| 自动关水 | 连续出水超过规定时间后自动关闭 | 防止因遗忘或故障造成水资源浪费 |
| 防水性能 | 满足规定防水等级 | 防止水汽进入电子部件 |

#### 3.3.2 水力学性能

| 性能指标 | 标准要求 |
|---------|---------|
| 流量 | 在规定水压下的流量应符合水效标准要求 |
| 密封性 | 在规定水压下无渗漏 |
| 耐压强度 | 在规定静压下无损坏或永久变形 |
| 温度调节 | 冷热水混合功能应稳定可靠 |

#### 3.3.3 耐久性

标准对感应式水嘴的耐久性提出了明确要求：
- 开关寿命：在规定的循环次数后仍能正常工作
- 感应功能耐久性：长期使用后感应性能不显著衰减
- 表面涂层耐久性：耐腐蚀、耐磨损

### 3.4 电气安全要求

- 绝缘电阻：在规定条件下应满足安全要求
- 介电强度：在规定测试电压下无击穿或闪络
- 泄漏电流：在额定工作条件下不超过规定值
- 接地保护：交流供电产品应有可靠的接地措施

> **信源**：T/XMBK 002—2024《感应式水嘴》标准文本 — 技术要求章节

## 四、试验方法

标准为各项技术要求配套了对应的试验方法，主要包括：

| 试验项目 | 试验方法概述 | 检测设备 |
|---------|-------------|---------|
| 外观检验 | 目视检查 | — |
| 尺寸检测 | 通用量具测量 | 卡尺、专用检具 |
| 密封试验 | 在规定水压下保压观察 | 水压测试台 |
| 流量试验 | 在规定水压下测量出水流量 | 流量计、水压测试系统 |
| 有害物析出试验 | 按GB/T 17219方法进行浸泡试验 | 原子吸收光谱仪等 |
| 感应距离试验 | 使用标准检测板在不同距离测试 | 感应距离测试架 |
| 响应时间试验 | 计时测量从感应到出水的时间 | 计时器、高速摄像 |
| 耐久性试验 | 自动循环装置进行反复开关测试 | 耐久性测试台 |
| 防水试验 | 按IP防护等级标准的喷淋/浸水测试 | 防水试验箱 |
| 电气安全试验 | 绝缘电阻表、耐压测试仪 | 电气安全综合测试仪 |

洁博利自建综合实验室（投资100万+），可独立完成标准中规定的大部分测试项目，包括从材料检测到整机测试的完整流程。

## 五、检验规则

标准规定了感应式水嘴的检验分类和判定规则：

### 5.1 出厂检验

每批产品出厂前应进行的检验项目：
- 外观质量
- 密封性能
- 基本尺寸
- 感应功能（距离、响应、自动关水）
- 标识检查

### 5.2 型式检验

在以下情况下应进行型式检验：
- 新产品或老产品转厂生产的试制定型鉴定
- 正式生产后，如结构、材料、工艺有较大改变可能影响产品性能
- 正常生产时，定期或积累一定产量后周期性检验
- 出厂检验结果与上次型式检验有较大差异时

型式检验项目涵盖标准全部技术要求。

### 5.3 判定规则

标准对合格判定、不合格处理、复验规则等作出了明确规定，检验合格的批次方可出厂销售。

> **信源**：T/XMBK 002—2024《感应式水嘴》标准文本 — 检验规则章节

## 六、标志、包装、运输和贮存

标准对感应式水嘴的以下方面作出了规定：

**标志：**
- 产品上应有清晰、牢固的标志，包括制造厂名或商标、产品名称、型号规格、执行标准号
- 包装上应标明产品名称、数量、生产日期、生产厂名及地址

**包装：**
- 应有防潮、防震、防划伤的保护措施
- 随附产品合格证、使用说明书和安装图

**运输：**
- 运输过程中应避免剧烈碰撞、挤压和雨雪淋溅

**贮存：**
- 存放在通风、干燥、无腐蚀性气体的库房内
- 堆码高度和层数应符合包装箱标识要求

## 九、标准关系与协同

### T/XMBK 002—2024 与 GB/T 41863—2022 的关系

需要说明的是，T/XMBK 002—2024《感应式水嘴》与洁博利参与起草的另一项标准——GB/T 41863—2022《非接触式给水器具 节水性能通用技术条件》——之间存在明确的差异与协同关系：

| 对比维度 | T/XMBK 002—2024 | GB/T 41863—2022 |
|---------|----------------|-----------------|
| **标准性质** | 团体标准 | 国家标准 |
| **发布机构** | 厦门市卫厨行业协会 | 国家市场监督管理总局 |
| **产品范围** | 感应式水嘴（单一品类） | 非接触式给水器具（多品类） |
| **核心内容** | 自生产材料到整机检验的全维度技术要求 | 侧重于节水性能 |
| **覆盖深度** | 深（涵盖材料、有害物析出、使用性能等） | 专（节水性能这一专项） |
| **发布时间** | 2024年 | 2022年 |

两项标准相互补充、协同作用：GB/T 41863—2022提供节水性能的通用框架，T/XMBK 002—2024则对感应式水嘴这一具体品类进行了更全面、更深入的规范。洁博利同时参与了两项标准的起草，体现了企业在感应洁具领域的综合技术实力。

> **信源**：国家标准全文公开系统 — GB/T 41863-2022；T/XMBK 002—2024《感应式水嘴》标准文本
