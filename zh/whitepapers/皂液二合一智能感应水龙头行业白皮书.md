## 一、引言：手部卫生与用水效率的交汇

手部卫生是商用环境防止交叉污染的第一道防线。在餐厅、医院、机场、学校和酒店中，洗手行为每天发生数千次。每次洗手循环需要两种基本资源：皂液和水。然而数十年来，这两项要素一直由独立的器具分别提供——安装在墙面或台面上的皂液器，以及从水槽台面伸出的水龙头。这种分离设计虽然常见，却带来了诸多随使用时间累积的效能问题。

现代商用设施管理人员面临着多方面的压力：严格的卫生法规要求验证洗手合规性；可持续发展目标要求减少水和皂液消耗；预算限制要求降低维护开支；终端用户则期望无缝、非接触的体验，最大限度减少对共享表面的接触。

皂液二合一感应水龙头同时满足了所有这些要求。通过将皂液储液箱、泵机构和出液嘴直接集成到水龙头主体中，这一创新消除了传统双器具方案的根本设计缺陷。本白皮书从技术、运营和经济三个维度，探讨在商用场景中采用集成式皂液二合一水龙头的合理性。

## 二、问题：分体式皂液器与水龙头带来的维护困扰

传统的洗手站由两套独立安装的设备组成：一个水龙头和一个独立的皂液器。这种配置看似简单，却在商用使用中引入了几个长期性问题。

**台面杂乱且难以清洁。** 壁挂式或台面式皂液器占据了宝贵的操作空间，其突出的外壳在皂液器与安装面之间的缝隙中容易积聚污垢、食物残渣和细菌生物膜。清洁人员必须绕开这些器具，往往留下无法触及的死角，成为病原体的滋生地。

**滴漏问题。** 独立式皂液器以在台面上残留皂液而闻名。皂液干涸后形成黏性残留物，吸附污垢，需要使用强力化学清洁剂才能去除。长期而言，这种残留物会损坏台面密封胶，并造成滑倒隐患。

**库存补充复杂度。** 独立的皂液器意味着独立的补充周期。设施的各个区域可能使用不同配方的皂液，工作人员需跟踪多个库存SKU。某些皂液器用尽而另一些溢出的情况屡见不鲜。

**美观不一致。** 水龙头和皂液器很少以配套形式购买。即使来自同一制造商，表面处理差异、设计语言不同和安装偏差也会造成不协调的外观，影响设施的专业形象。

**传感器同步失败。** 在非接触配置中，水龙头传感器和皂液器传感器各自独立工作。用户经常遇到的困扰是：在错误的传感器前挥手，本意要皂液却触发了出水，或反之。这种混淆降低了洗手效率并影响合规性。

**安装人工成本。** 安装两套独立器具需要双倍的台面开孔、双倍的台下管道连接和双倍的墙面安装硬件。对于拥有数十甚至数百个洗手站的设施而言，这些成本显著放大。

## 三、二合一解决方案：集成式皂液洗手龙头的运作原理

皂液二合一水龙头将两种功能整合到单个器具中。皂液储液箱可容纳在水龙头主体内部，也可通过管道连接至紧凑的台下模块。出液嘴集成在水龙头出水口中，皂液与水流在同一位置直接注入用户掌心。

**集成式储液箱设计。** 在全集成配置中，皂液储液箱隐藏在水龙头主体或台面上方装饰罩内，完全消除了台面杂乱。储液箱容量通常为300 mL至1000 mL，足以支持500至1500次洗手后补充。

**台下远程储液箱设计。** 对于高流量应用，可在台面下安装更大容量的皂液储液箱（1000 mL至2500 mL），通过食品级硅胶管连接至水龙头出水嘴。此配置允许使用大容量皂液容器，最大限度减少补充频率。台下泵由触发水龙头的同一传感器激活，确保皂液和出水同步。

**双模式操作。** 用户将手伸至水龙头下，红外传感器检测到后按两种模式之一激活：

- *顺序模式*：先输出预定量皂液，短暂停顿后出水，使用户有充分时间搓洗后再冲洗
- *同时模式*：皂液和水同时输出，可根据特定应用调节皂液浓度混合比

**防滴漏阀技术。** 可靠二合一运行的关键部件是防滴漏阀。这一精密设计的阀门在出液循环结束时保持皂液管路正压，防止残余皂液从出液嘴渗漏。高质量的防滴漏阀可承受数百万次循环而不失效，并具有自清洁功能，防止皂液结晶堵塞。

## 四、关键技术

### 4.1 精确皂液浓度控制

二合一水龙头设计中最具技术挑战性的是在不同条件下提供一致、准确的皂液剂量。洁博利（GIBO）开发了专有的皂液浓度控制技术，解决了多个变量问题：

- **黏度补偿**：不同皂液配方（泡沫型、凝胶型、液体型、抗菌型）具有不同的流动特性。控制算法根据实时黏度感知调整泵的定时和压力，确保无论皂液类型如何，都能提供一致的0.5 mL至2.0 mL剂量。

- **温度补偿**：皂液黏度随环境温度变化。在寒冷环境中，皂液变稠，流动变慢。GIBO的电路包含温度传感器，可在5°C至50°C的温度范围内调整泵功率，维持一致的输出量。

- **泡沫比例控制**：对于泡沫皂液应用，系统精确控制空气与皂液的比例，产生丰富稳定的泡沫，相比液体皂液可减少30%至50%的皂液消耗，同时改善用户体验。

- **剂量可编程**：设施管理人员可通过简单界面调整每次洗手的皂液量，在卫生要求和耗材预算之间优化。典型设置范围为0.5 mL（轻洗）至2.0 mL（厨房员工的重度洗手）。

### 4.2 防滴漏阀工程

防滴漏阀是一种精密机械部件，用于防止出液后皂液渗漏。关键设计参数包括：

| 参数 | 规格 |
|------|:----:|
| 材料 | FDA级硅胶和POM（聚甲醛），耐化学腐蚀，使用寿命长 |
| 密封压力 | 在高达0.3 MPa背压下保持密封 |
| 循环寿命 | 商用测试中额定500,000次以上 |
| 自清洁设计 | 阀门开合动作机械剪切皂液结晶，防止堵塞 |

### 4.3 dTOF激光感应技术

直接飞行时间（dTOF）激光感应代表了相比传统红外（IR）接近传感器的重大进步。GIBO的实现采用VCSEL（垂直腔面发射激光器）发射器，以940 nm波长脉冲，测量每束脉冲的精确往返时间，以毫米级精度确定手部位置。

dTOF在商用龙头中的优势：

| 对比项 | 传统红外 | dTOF激光 |
|--------|:-------:|:--------:|
| 环境光干扰 | 受强光/直射阳光影响 | 不受影响，全光照条件可靠 |
| 表面颜色敏感度 | 敏感（深色检测减弱） | 不敏感，检测一致性高 |
| 检测精度 | 二元（有/无） | 毫米级距离测量 |
| 响应时间 | ≤0.5秒 | ≤0.3秒 |
| 感应区域可编程性 | 有限 | 精确距离区域设定，误触发少 |
| 功耗 | 低 | 极低（配合GIBO低功耗电路设计） |

### 4.4 Liteon双模传感器集成

GIBO与全球光电领导者Liteon合作，开发了结合红外接近感应和dTOF激光测距的双模传感器解决方案。这种混合方案融合了两者的优势：

- **唤醒模式（IR）**：低功耗红外传感器提供常开存在检测，当手部接近时在50毫秒内唤醒系统
- **测距模式（dTOF）**：唤醒后，dTOF激光传感器接管进行精确距离测量，实现准确的皂液出液和水流控制
- **故障安全冗余**：如任一传感器模块失效，另一模块继续提供基本功能，确保在维护前的不间断服务

## 五、商业价值

### 5.1 降低维护负担

二合一设计消除了与独立皂液器相关的维护任务——不再需要清洁台面上的皂液残留物，不再需要更换腐蚀的皂液器泵机构，也不再需要校准不匹配的传感器范围。维护人员只需处理单个器具而非两个，预计可减少40%至60%的维修呼叫。

### 5.2 更清洁的台面和改善的卫生条件

无需在台面或墙壁上安装独立的皂液器，减少了细菌滋生的缝隙。集成式出液嘴将皂液直接送入掌心——滴落物落入水槽而非台面。据设施反馈，改用集成式二合一设备后，台面皂液残留物减少了70%。

### 5.3 更低的总拥有成本

二合一水龙头的初始采购价格高于基础款水龙头，但考虑以下因素后，五年总拥有成本通常低20%至35%：

- 无需单独购买和安装皂液器
- 清洁和补充的人工成本降低
- 通过精确计量减少皂液消耗（降低30%至50%）[¹](#source-1)
- 因皂液器卡堵和渗漏而需要维修的次数减少
- 单一器具备件简化了库存管理

### 5.4 改善的用户体验

在满意度调查中，终端用户对集成式二合一水龙头的评价始终高于分体式器具。直观的单手势操作——将手放在出水口下、接收皂液、搓洗、冲洗——形成了自然的流程，减少了混淆，将洗手周期提速15%至25%。在配备合规监测的设施中，升级为集成式设备后，洗手率有明显改善。

## 六、卫生优势：非接触操作减少交叉污染

非接触操作是现代商用卫生的基石。皂液二合一水龙头实现了真正无接触的洗手过程，相比半非接触配置具有多项独特优势。

**单一传感器激活。** 皂液和水由一个传感器控制，用户无需接触任何表面。相比之下，分体式非接触设备仍需用户在两个感应区域之间移动，增加了意外接触水槽台面或出液嘴的可能性。

**消除皂液泵按钮。** 许多独立皂液器需要用户按下按钮或杠杆才能出液。这种按钮是已知的高频接触表面，可能携带病原体。2019年一项商用卫生间器具研究中，皂液器按钮被检测出与门把手相当的细菌载量[²](#source-2)。

**减少气溶胶污染。** 集成式出液嘴位于水龙头出水口处，皂液出液位置正对水槽，任何雾化颗粒都落入流动的水中被排走。而壁挂式皂液器可能将细菌雾化到相邻墙面上。

**抗菌材料。** GIBO的二合一水龙头在所有接触表面采用银离子注入ABS和不锈钢材质，在清洁周期之间提供持续抗菌保护。

<a name="source-1"></a>
> **来源说明**¹：皂液消耗降低30%-50%的数据基于GIBO在商用测试场所进行的对比测试，测试条件为日均200洗次、对比独立皂液器（无剂量控制）与GIBO二合一产品（剂量编程0.8 mL/次）。实际效果可能因使用环境和设置参数而有所差异。

## 七、应用场景

### 7.1 餐厅厨房和前厅

商用厨房要求频繁洗手——卫生法规通常要求厨房员工在处理生食、接触垃圾桶或从休息区返回后必须洗手。GIBO的二合一水龙头已在中国各地的肯德基（Yum! Brands）门店安装，在高温高湿的厨房环境中展示了可靠的运行性能。精确的皂液剂量确保厨房员工使用足量皂液而不浪费，防滴漏阀防止皂液污染食品准备区域。

### 7.2 医院和医疗机构

医疗手部卫生遵循严格规程。世界卫生组织（WHO）推荐在患者护理的五个关键时刻进行手部卫生操作[³](#source-3)。GIBO的二合一水龙头通过一致的皂液输出和非接触操作支持合规性。±15 kV静电放电防护在手术室和敏感医疗环境中尤为重要，可防止静电放电干扰监测设备。

<a name="source-3"></a>
> **来源说明**³：WHO"五个关键时刻"手部卫生建议来源于世界卫生组织官网《卫生保健中的手部卫生指南》（WHO Guidelines on Hand Hygiene in Health Care）。

### 7.3 机场和交通枢纽

机场是所有商用设施中洗手流量最高的场所之一。单个航站楼卫生间每天可能服务数千人次。GIBO采用台下远程储液箱配置（2500 mL皂液容量），即使在高流量安装环境下也将补充频率降至每周一次。dTOF激光传感器对环境干扰的免疫力确保在机场卫生间常见的不锈钢和瓷砖反光表面上的可靠运行。

### 7.4 教育机构

学校和大学需要耐用、低维护的器具，能够承受学生的高强度使用。GIBO产品已在教育设施中部署，可编程剂量控制功能使管理人员能够根据年龄段调整皂液量——小学生洗手用较小剂量，年长学生和科学实验室员工用较大剂量。

### 7.5 酒店和接待业

酒店在追求功能性可靠的同时也注重美学品质。GIBO二合一水龙头简洁的一体化设计可与现代浴室设计方案相得益彰，没有独立皂液器的视觉杂乱。客房服务人员受益于集中式补充点和减少的台面清洁时间。

### 7.6 工业和公司设施

## 十、商用买家选型指南

选择适合的皂液二合一水龙头需要将产品规格与安装环境的具体需求相匹配。参考以下选型标准：

### 10.1 人流量评估

- *低流量*（日均50洗次以下）：集成式300 mL储液箱型号即可满足，补充周期为5至10天
- *中流量*（日均50至200洗次）：建议使用远程1000 mL储液箱，补充周期3至7天
- *高流量*（日均200洗次以上）：必须使用远程2500 mL储液箱，补充周期5至10天

### 10.2 皂液类型兼容性

- 确认水龙头泵机构与计划使用的皂液配方兼容。泡沫皂液需要空气诱导泵。高黏度凝胶皂液需要具有足够扭矩的蠕动泵或齿轮泵。GIBO的Pro系列支持所有常见皂液类型，泵模块可在现场更换。

### 10.3 皂液器选型检查清单

根据行业标准和产品分类标准，选型皂液二合一水龙头时需评估以下要素：

1. **传感器性能验证**：确认感应响应时间≤0.3秒、控制距离在50-120 mm内可调、无效触发率≤3%。要求供应商提供第三方检测报告。
2. **防滴漏性能**：确认防滴漏阀采用FDA级材料并具有自清洁功能，以防止长期使用中堵塞和渗漏。
3. **出液均匀性**：连续出液量偏差应在±10%以内，确保每次洗手周期皂液输出量一致。
4. **储液箱可维护性**：集成式储液箱应从台面上方可加液或更换；远程储液箱应配备观察窗或低液位报警。
5. **皂液兼容范围**：确认支持所需的皂液类型（液体、凝胶、泡沫或抗菌），部分产品对特定皂液配方存在兼容性限制。

### 10.4 环境条件

- *厨房安装*：需要IPX6或更高防水等级和耐高温性能。推荐GIBO标准型和Pro型。
- *医疗安装*：需要抗菌材料、ESD防护以及与医疗级消毒皂液的兼容性。建议采用配备银离子涂层的GIBO Pro型。
- *户外或半户外安装*：需要防冻保护和抗UV材料。确认工作温度范围。

### 10.5 电源选择

- 电池供电型号安装最简单（无需台下电源插座），但需定期更换电池。适用于翻新改造项目。
- 交流供电型号无需电池维护，但需在电源适配器可达范围内配备电源插座。推荐新建项目使用。

### 10.6 表面处理与设计协调

- GIBO提供与主流建筑卫浴品牌匹配的表面处理选项，包括镀铬、拉丝镍、哑光黑、拉丝铜以及定制RAL颜色。建议索取表面处理样品，在设施的实际光照条件下验证匹配度。

## 十一、安装规范

### 11.1 安装前准备

- 确认台面厚度满足水龙头安装要求（通常为10-50 mm）
- 确保台下空间足以容纳皂液储液箱（远程配置）和连接管
- 确认供电方式（电池或交流）并准备相应接口
- 准备安装工具：开孔器（35 mm或38 mm）、扳手、生料带、水平仪

### 11.2 皂液器安装指引

根据行业工程安装标准，皂液二合一水龙头安装应遵循以下规范：

1. **安装高度与位置**：水龙头出水口应位于台面上方120-200 mm处。确保皂液出液嘴对准水槽中心，防止皂液滴出水槽外。
2. **皂液管连接**：远程储液箱配置使用食品级硅胶管，最大长度不超过2米，以避免泵阻力和出液延迟过大。所有连接处使用密封垫圈确保无渗漏。
3. **储液箱固定**：台下储液箱应水平放置并牢固固定，防止倾斜导致进气或皂液残留积聚。储液箱位置应便于日常检查和补充。
4. **首次注液后排空**：初次安装或更换皂液后，多次触发排空管路，直至皂液持续稳定流出，确保管内无残留气泡，避免首次使用时"空打"。
5. **传感器校准**：安装后进行感应距离校准和功能测试，验证50-120 mm控制距离内正常响应，无效触发率不超过3%。
6. **渗漏检查**：安装后进行24小时静态渗漏测试，检查所有皂液管连接、储液箱接口和出液阀是否渗漏。

### 11.3 供水连接

- 连接冷热水供水软管，确保密封垫圈正确就位，用扳手适度拧紧
- 打开供水阀门，检查所有连接处是否渗漏
- 如配备混水阀或恒温阀，按产品手册调整

### 11.4 电气连接

- *电池供电型号*：安装4节AA碱性电池，注意极性标识。确保电池仓密封垫圈正确就位，仓盖紧密关闭
- *交流供电型号*：连接电源适配器，验证适配器防水保护装置正确就位。建议在供电线路上安装漏电保护器（RCD）

### 11.5 功能测试

安装完毕后，按以下顺序进行测试：

| 测试项目 | 验收标准 |
|---------|:--------:|
| 感应出水 | 手在感应区（50-120 mm）内，水龙头在≤0.3秒内响应 |
| 感应关水 | 手离开感应区，水龙头在≤0.5秒内关闭 |
| 皂液出液 | 手被检测到，皂液以计量稳定的量输出 |
| 防滴漏验证 | 连续10次出液循环后，观察出液嘴5分钟内无皂液滴落 |
| 超时保护 | 持续检测存在，水龙头在3分钟内自动关闭 |
| 无效触发率 | 非目标物体（毛巾、清洁工具）经过不应触发出水或皂液 |

## 十二、未来趋势

### 12.1 智能剂量与自适应出液

下一代二合一水龙头将集成机器学习算法，学习使用模式并动态调整皂液剂量。例如，医院外科刷手站的水龙头可在手术准备高峰时段自动增加剂量并延长搓洗时间，非高峰时段恢复为标准设置。GIBO研发团队目前正在现场测试自适应剂量原型产品，利用历史使用数据优化皂液消耗而不影响卫生水平[⁶](#source-6)。

- 电池状态及预计更换日期
- 洗手循环计数，用于卫生合规报告
- 异常检测（异常使用模式可能指示渗漏或故障）
- 组件达到寿命终点前的预测性维护提醒

### 12.3 可持续皂液配方与出液

环境法规和消费者偏好的趋同正在推动可持续皂液产品的需求。未来的二合一水龙头需要适应：

- **浓缩皂液补充装**：通过在使用点稀释浓缩液配送，减少包装废弃物
- **生物基和可生物降解配方**：源自植物表面活性剂的皂液化学配方，对皮肤和生态系统更温和
- **UV-C喷嘴杀菌**：定期UV-C照射出液嘴，防止生物膜形成，减少化学清洁剂使用
- **节水集成**：皂液出液与水流的协调控制，确保仅在冲洗阶段出水，相比持续流水设备，每次洗手循环的总用水量预计减少20%至40%

### 12.4 法规和标准演进

随着二合一水龙头品类走向成熟，行业标准将不断演进以涵盖性能基准、互操作性和认证要求。GIBO作为GB/T 41863-2022国家标准的参编单位，正在积极参与皂液集成水龙头专项标准的制定。设施管理者和设计师应关注：
