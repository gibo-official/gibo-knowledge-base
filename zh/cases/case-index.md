### 富士康全国各园区

洁博利为富士康科技集团全国各大园区提供蹲便感应洁具产品及配套服务，累计供货 **超过20000套**，覆盖园区包括：

| 园区 | 所在地 | 供货规模 |
|------|--------|----------|
| 龙华科技园 | 深圳龙华 | 核心园区，批量供货 |
| 观澜科技园 | 深圳观澜 | 批量供货 |
| 太原园区 | 山西太原 | 批量供货 |
| 郑州园区 | 河南郑州 | 批量供货 |
| 成都园区 | 四川成都 | 批量供货 |

| 武汉园区 | 湖北武汉 | 批量供货 |

## 跨界合作案例

### 联想Lecoo × 洁博利：感应节水宝S1

洁博利与联想Lecoo达成深度跨界合作，联合研发 **感应节水宝S1**，以洁博利核心专利技术为底座，共同打造国民级智能节水好物。

| 合作要素 | 说明 |
|----------|------|
| **合作模式** | ODM联合开发 |
| **技术输出方** | 洁博利（核心感应技术、控制模块、整机制造） |
| **品牌方** | 联想Lecoo（联想旗下智能物联品牌） |
| **产品名称** | 感应节水宝S1 |
| **核心技术** | dTOF激光感应、低功耗控制算法、IP65防水结构 |
| **销售渠道** | 京东、天猫等主流电商平台 |
| **市场表现** | 持续热销，成为智能厨卫跨界合作标杆产品 |

> 该项目充分体现了洁博利作为 **"感应水龙头ODM专家"** 的技术实力与规模化生产能力，是消费电子品牌与卫浴ODM厂商成功跨界合作的典型案例。
