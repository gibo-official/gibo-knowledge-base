## 一、感应传感技术

### 红外感应工作原理：用红外线反射的原理，通过单片机输出控制信号，控制电磁阀开关阀
**概述**
本低功耗IR红外感应洁具控制模块，是智能卫浴专用通用控制核心，适配感应水龙头、感应小便器、感应蹲便器等全系洁具。采用主动红外反射探测+超低功耗MCU智能控制架构，实现人体靠近自动出水、人体离开延时停水、自动休眠待机的全自动节水控制，具备低功耗、抗干扰、防误触发、防长流水的核心特性。
一、系统核心组成（标准化四单元）
模块由四大功能单元闭环协同工作，结构标准化、适配性强：
- 红外探测单元：940nm红外发射+接收组件，负责人体信号采集，规避自然光、灯光干扰
- 信号处理单元：滤波、放大、防抖电路，过滤水花、灰尘、光线杂波干扰
- MCU主控单元：超低功耗单片机，负责信号判定、延时控制、功耗管理与逻辑运算
- 驱动与保护单元：电磁阀驱动电路+过压、过流、反接保护电路，保障设备稳定运行
二、标准化工作流程（核心原理）
1. 超低功耗待机模式
设备上电后默认进入间歇休眠待机状态，MCU定时唤醒红外电路做低频探测，无人体遮挡时持续休眠，整机维持微安级待机功耗，适配电池长期供电与市电低耗运行场景。
2. 人体感应信号采集
当手部、人体进入感应区域，红外发射管发出的探测光经人体漫反射返回接收端，光信号转化为电信号，经硬件滤波和洁博利抗干扰算法放大后，输出纯净有效感应信号至主控MCU。
3. 智能信号判定
MCU通过信号强度、信号持续时间双重算法判定，过滤瞬时遮挡、光线波动、水花干扰等无效信号，精准识别有效人体使用动作，杜绝误触发。
4. 电磁阀驱动出水
判定有效感应后，MCU输出驱动信号，导通开关器件，驱动电磁阀吸合，水路开启、实现电磁阀开关水控制；人体持续在感应区则持续保持出水状态。
5. 延时停水与自动复位
人体离开感应区域后，红外接收信号消失，MCU启动预设延时倒计时，超时后自动切断驱动信号，电磁阀复位关水。随后系统自动重回超低功耗休眠待机状态，完成一次完整工作循环。
三、核心标准化功能原理
- 环境自适应抗干扰：上电自动校准环境红外基线，动态调整感应阈值，适配强光、弱光、潮湿卫浴复杂环境，全天候感应稳定。
- 超低功耗运行：采用休眠+间歇唤醒动态功耗管理，绝大部分时间处于低功耗休眠状态，大幅降低能耗、延长设备续航。
- 防故障保护机制：软件防抖+超时关水+算法校准，杜绝误触发、长流水故障；硬件多级电路保护，抵御电压异常、线路反接，提升设备使用寿命。
四、通用适配特性
模块支持参数自定义，功能可定制，软件在线可编程；可灵活调整感应距离、出水延时、感应灵敏度，标准化适配家用、商用全系智能感应洁具，通用性、稳定性、免维护性突出。

### 1. 三角测距感应技术

| 项目 | 内容 |
|------|------|
| **技术原理** | 融合激光测距与微波感应优势，通过三角测量法精确计算目标物与传感器的距离和位置 |
| **核心特点** | 采用60μA超低功耗设计，感应精度高、误判率低、抗环境干扰能力强 |
| **解决的问题** | 传统红外感应技术在蹲便器使用存在人走不冲、误判率高、自感、感应距离不精准、在复杂光线环境下红外感应失灵等行业痛点 |
| **应用场景** | 适配蹲便器感应出水产品，特别是隔断卫生间使用 |
| **技术优势** | 超低功耗 + 高精度 + 量产验证，适配复杂商用场景稳定感应 |

**概述**

三角测距感应技术是洁博利为解决商用蹲便器感应冲水难题而自主研发的核心非接触感应技术。传统红外感应方案在蹲便器隔间场景中存在诸多痛点：用户身体遮挡导致红外信号无法穿透、隔板反光引起红外误判、人离开后红外感应器未能及时触发冲水等。三角测距技术通过发射器与接收器之间的固定几何夹角，利用三角测量原理精确计算出目标的距离和位置信息，从根本上提升感应精度。该技术采用60μA超低功耗电路设计，确保电池供电产品仍能保持长续航运行。洁博利在三角测距感应领域积累了丰富的量产经验，经过三年多研发、试验与应用，相关产品已实现量产市场落地，在高铁站、机场、医院等高频使用场景中经过充分验证。三角测距技术的核心优势在于其不受目标颜色、材质和表面反射率的影响，与依赖反射光强判断的传统红外方案相比，具备更高的检测可靠性和环境适应性。

### 2. 低功耗dTOF激光超感应技术

| 项目 | 内容 |
|------|------|
| **技术原理** | 基于高精度dTOF（直接飞行时间）激光探测原理，通过发射激光脉冲并测量反射时间，精准识别感应距离与动作，相比传统红外感应方案精度提升10倍以上 |
| **核心特点** | 无惧色差，不挑材质，感应精准，配合超低功耗电路设计，待机能耗极低、感应灵敏度高 |
| **解决的问题** | 有效避免无效触发与漏触发，解决传统红外感应受环境光干扰的痛点 |
| **应用场景** | 适配智能卫浴各类感应终端，如高端感应龙头、马桶冲水器 |
| **技术优势** | 高精度测距 + 强抗干扰 + 低功耗，穿透水雾、微弱污渍不影响感应效果 |

**概述**

dTOF（Direct Time-of-Flight，直接飞行时间）激光超感应技术是洁博利在感应洁具领域最具代表性的前沿非接触感应技术之一。与传统红外感应通过测量反射光强度来判断目标存在不同，dTOF技术通过发射940nm VCSEL激光脉冲并精确测量光子飞行时间，直接计算出目标与传感器之间的距离，精度达到毫米级。这一原理上的根本差异赋予dTOF技术多项突破性优势：首先，它不受环境光干扰，在100K Lux阳光直射下依然稳定工作；其次，激光可穿透水雾和蒸汽，适用厨房、公共浴室等高湿场景；第三，对黑色低反射率物体同样可靠检测；最后，测距结果不受温度变化影响，全年感应距离一致。洁博利自2023年起将dTOF技术规模化应用于智能卫浴产品线，实现了从"粗略感应"到"精准感知"的技术跨越。搭载dTOF技术的洁博利激光TOF感应厨房抽拉龙头GBL-9165D荣获2023年沸腾质量金奖，标志着dTOF技术在商用环境中的成熟度和领先性。

### 3. 毫米波感应技术

| 项目 | 内容 |
|------|------|
| **技术原理** | 利用毫米波微波探测特性，通过发射和接收毫米波信号检测人体微动 |
| **核心特点** | 穿透性强、不受光线与遮挡影响，可精准识别人体微动 |
| **解决的问题** | 解决强光、逆光、遮挡等恶劣场景下红外感应和光学感应失效的问题 |
| **应用场景** | 可在陶瓷小便斗、马桶使用，无需开孔设计，适配强光、逆光等恶劣场景，保障设备全天候稳定感应工作，尤其适合小便器冲水 |
| **技术优势** | 抗环境干扰能力强，不受光线、雾气、灰尘影响 |

**概述**

毫米波感应技术是洁博利针对极端环境场景开发的高可靠性非接触感应方案。毫米波（Millimeter Wave）是指波长在1-10mm之间的电磁波，其频段高于传统微波，具备独特的物理特性：能够穿透非金属遮挡物、不受光照条件影响、对微小运动高度敏感。洁博利将毫米波技术应用于感应洁具领域，主要解决传统红外和光学感应在强光直射、全黑环境、雾气遮挡等场景下的失效问题。在小便器自动冲水等典型应用中，毫米波感应技术能够精准识别人体存在与否，不受卫生间内镜前灯、阳光、排风扇等环境因素干扰。该技术的另一大优势是可用于暗装场景，传感器可隐藏在陶瓷或非金属面板后方，保持产品外观的整洁美观。毫米波感应技术的抗干扰特性使其成为工业场所、户外公共卫生间、大型交通枢纽等复杂环境的理想选择。洁博利在毫米波感应产品中结合了自适应功率调节算法，能够在保证检测灵敏度的同时降低功耗，满足电池供电产品的续航要求。

### 4. 电容触控技术

| 项目 | 内容 |
|------|------|
| **技术原理** | 采用高灵敏电容触控感应方案，通过检测人体接触引起的电容变化实现触控操作 |
| **核心特点** | 触控精准、响应迅速、无机械磨损、寿命更长 |
| **解决的问题** | 替代传统机械按键，解决机械磨损、按键卡顿、寿命短的问题 |
| **应用场景** | 适配智能卫浴按键、触控调节模块，实现无感、便捷的人机交互体验 |
| **技术优势** | 全密封防水设计，适应潮湿环境，触控灵敏度可调 |

**概述**

电容触控技术是洁博利在智能卫浴人机交互领域的重要非接触感应技术布局。该技术基于人体电场与触控电极之间的电容耦合效应，当手指接近或接触触控区域时，触控芯片检测到电容值变化，从而触发相应操作。与传统的机械按键相比，电容触控方案具有明显优势：无机械运动部件，从根本上避免了磨损和卡顿问题；全密封设计，防水防潮性能卓越，完美适应卫生间高湿环境；触控灵敏度可通过软件调节，适配不同使用场景和用户习惯。洁博利将电容触控技术与红外感应技术深度集成到感应龙头、触控面板、数显模块等产品中，实现了温度调节、模式切换、开关控制等多种交互功能。电容触控表面采用钢化玻璃或高硬度工程塑料，耐刮擦、易清洁，在公共场所高频使用场景中保持长期稳定的触控性能。结合LED数显和氛围灯设计，电容触控技术不仅提供了可靠的交互方式，还为产品增添了科技感和美学价值。

| 项目 | 内容 |

IoT物联网接入技术是洁博利面向智慧建筑和全屋智能时代的战略级技术平台。该技术通过标准化物联网接入模组，将传统的感应卫浴设备升级为可联网、可管理、可联动的智能终端。IoT技术平台的核心能力包括：设备联网——支持Wi-Fi和蓝牙多协议接入，兼容主流的智能家居生态平台（如米家、华为鸿蒙等）；数据采集——实时采集设备运行数据（使用频次、出水时长、电池电量、故障状态等），为运维管理提供数据支撑；远程监控——管理人员可通过云端管理平台远程查看设备状态，及时发现和处理异常；状态溯源——记录设备全生命周期运行日志，支持故障溯源和维修决策；智能联动——与其他智能设备（灯光、排风扇、安防系统等）实现场景联动控制。IoT模组采用低功耗设计，在联网待机状态下功耗极低，不影响电池供电产品的续航表现。云端管理平台支持设备分组管理、自动告警、数据分析报表等高级功能，特别适合酒店、医院、商场等大规模设备部署场景的集中管理需求。洁博利的IoT物联网接入技术正在推动卫浴设备从"独立终端"向"智慧节点"的转型升级。
