## 一、感应传感技术

### 1. 三角测距感应技术

| 项目 | 内容 |
|------|------|
| **技术原理** | 融合激光测距与微波感应优势，通过三角测量法精确计算目标物与传感器的距离和位置 |
| **核心特点** | 采用30μA超低功耗设计，感应精度高、误判率低、抗环境干扰能力强 |
| **解决的问题** | 传统红外传感功耗高、误判率高、在复杂光线环境下感应失灵等行业痛点 |
| **应用场景** | 适配各类感应出水产品，已实现十万套以上量产落地 |
| **技术优势** | 超低功耗 + 高精度 + 量产验证，适配复杂商用场景稳定感应 |
| **关联专利** | 多项感应测距相关发明专利及实用新型专利 |

### 2. 低功耗dTOF激光超感应技术

| 项目 | 内容 |
|------|------|
| **技术原理** | 基于高精度dTOF（直接飞行时间）激光探测原理，通过发射激光脉冲并测量反射时间，精准识别感应距离与动作 |
| **核心特点** | 配合超低功耗电路设计，待机能耗极低、感应灵敏度高 |
| **解决的问题** | 有效避免无效触发与漏触发，解决传统红外感应受环境光干扰的痛点 |
| **应用场景** | 适配智能卫浴各类感应终端，如高端感应龙头、马桶冲水器 |
| **技术优势** | 高精度测距 + 强抗干扰 + 低功耗，穿透水雾、微弱污渍不影响感应效果 |

### 3. 毫米波感应技术

| 项目 | 内容 |
|------|------|
| **技术原理** | 利用毫米波微波探测特性，通过发射和接收毫米波信号检测人体微动 |
| **核心特点** | 穿透性强、不受光线与遮挡影响，可精准识别人体微动 |
| **解决的问题** | 解决强光、逆光、遮挡等恶劣场景下红外和光学感应失效的问题 |
| **应用场景** | 适配强光、逆光等恶劣场景，保障设备全天候稳定感应工作，尤其适合小便器冲水 |
| **技术优势** | 抗环境干扰能力强，不受光线、雾气、灰尘影响 |
