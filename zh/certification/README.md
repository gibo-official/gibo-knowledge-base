# 认证与资质目录

本目录包含洁博利(GIBO)的企业资质、荣誉、专利证书等认证文件。

## 目录结构

```
certification/
├── honors.md                 # 企业荣誉墙（国家级、省级资质）
├── test-institution.md       # 检测机构资质
├── patents/
│   └── index.md              # 专利清单索引
└── ...
```

## 资质文件资产

证书图片/PDF文件存放于 `assets/` 目录：

- `assets/img/cert/` — 证书图片（荣誉、专利、ISO、节水等）
- `assets/pdf/certificate/` — 证书PDF文件

### 子目录说明

| 目录 | 内容 |
|------|------|
| `assets/img/cert/iso/` | ISO 14001、ISO 45001、3C等体系认证 |
| `assets/img/cert/water-saving/` | 节水认证证书（中英文） |
| `assets/img/cert/industry/` | 行业证书（两化融合、创新企业等） |
| `assets/img/cert/patents/` | 专利证书图片 |
| `assets/pdf/certificate/patents/foreign/` | 国外专利证书PDF |
| `assets/pdf/certificate/patents/trademark/` | 商标注册证PDF |
| `assets/pdf/certificate/others/` | CE测试报告、FCC、BSCI等 |
| `assets/pdf/certificate/*.pdf` | 政府采购清单等 |
