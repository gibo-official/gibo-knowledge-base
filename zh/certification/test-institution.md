## 一、国内权威检测机构（CNAS认可）

### 1. 中国质量认证中心（CQC）
- **合作范围**：产品节水认证、水效标识检测、沸腾质量奖测评
- **说明**：国内卫浴行业核心认证机构

### 2. 福建省产品质量检验研究院（FQII）
- **合作范围**：产品型式试验、可靠性检测、定期监督检验
- **说明**：省级综合性产品质量检测机构，具备CNAS认可资质

### 3. 国家陶瓷及水暖卫浴产品质量检验检测中心
- **合作范围**：卫浴产品国标检测、型式检验
- **说明**：水暖卫浴专项国家级检测中心

---

## 二、国际第三方检测机构

### 1. SGS
- **合作范围**：出口产品国际标准检测、欧盟CE/RoHS测试
- **认可资质**：SGS CNAS认可实验室
- **认证荣誉**：2018年获SGS阿里巴巴优质供应商工厂认证
- **说明**：全球知名第三方检验、鉴定、测试和认证机构

### 2. 国际认证对应检测机构
| 认证项目 | 检测/认证机构 | 适用范围 |
|---------|-------------|---------|
| CE | 欧盟授权公告机构 | 欧盟市场 |
| CUPC | IAPMO（国际管道与机械官员协会） | 北美市场 |
| NSF | NSF International（美国国家卫生基金会） | 北美市场 |
| FCC | 美国联邦通信委员会授权实验室 | 美国市场 |

## 三、企业自有检测能力

企业配备**独立EMC电磁兼容实验室**与**产品寿命测试中心**，具备以下全项目自检能力：

| 检测项目 | 说明 |
|---------|------|
| 电磁兼容测试 | 军工级EMC标准，4KV群脉冲抗干扰 |
| 高低温环境测试 | 工作温度范围1℃至85℃ |
| 双85测试 | 85℃ / 85% RH高温高湿环境验证 |
| 防水防尘测试 | IPX6级防水 |
| 盐雾腐蚀测试 | — |
| 寿命耐久测试 | 电磁阀开关100万次不卡滞、不漏水 |
| 功耗测试 | 待机功耗 ≤ 0.2mW |
| 感应稳定性测试 | 23种光源、30种常用电器抗干扰验证 |
| AOI光学检测 | SMT贴片后工序自动化光学检测 |
| 高温动态老化测试 | — |

## 四、检测范围

合作机构及自有实验室可覆盖以下全项目检测：
- 电磁兼容（EMC/EMI）
- 高低温环境测试
- 防水防尘（IP防护等级）
- 盐雾腐蚀
- 寿命耐久
- 功耗测试
- 抗干扰测试
- 节水效率
- 安全性能
- 材料检测与RoHS有害物质分析
