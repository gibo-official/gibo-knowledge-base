# 单窗双感应模块

**文档版本**：V2.0
**最后更新**：2026-06-12
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 单窗双感应模块是洁博利拥有国家发明专利的核心感应组件，在单一感应窗口内集成接近感应与温控感应双功能，实现感应出水与数显温控的一体化设计，为行业首创技术方案。

## 方案概述

单窗双感应模块是洁博利自主研发的核心技术创新成果，依托国家发明专利（发明专利，2020年授权）进行产业化落地。该方案的诞生源于对高端卫浴市场"更简洁、更智能"趋势的深刻洞察——传统感应龙头需要在表面开设两个独立的感应窗口，分别用于接近感应和温控调节，不仅增加了结构复杂度，也影响了龙头整体的美观性。

洁博利研发团队经过多年技术攻关，在行业内首次提出"单一感应窗口内集成接近感应与温度感应双功能"的创新方案。该模块采用创新的光学窗口设计，将一个感应窗口精细划分为两个功能区：感应区负责接近感应（检测手部靠近信号，实现自动出水），温控区负责温度感应（通过触控方式实现水温调节）。两个区域在物理结构上共用一个窗口盖板，但在光学通道和信号链路上完全独立。

在信号处理层面，模块搭载自研双通道信号处理系统，两个感应在电路层面并行运行，互不干扰。自研专利算法负责信号隔离与识别——当用户靠近感应区时，系统快速识别接近意图并启动出水；当用户触摸温控区时，算法精准区分触控操作与接近信号，确保两种操作指令不串扰、不漏判，识别准确率经过严格测试验证。

最终实现的是一体化设计效果：感应出水与数显温控融合在同一个感应窗口中，无需在龙头表面开设两个感应窗，外观更加简洁流畅。模块可配合LED数显温控模组，实时显示当前水温，为用户提供直观的交互反馈。该方案已成功应用于洁博利多款旗舰产品，成为高端感应龙头领域的技术标杆方案。

## 功能特点

- **行业首创单窗双感应**：单一窗口同时实现接近感应与温度感应两种功能，为卫浴行业创新设计，解决了传统双窗方案的结构冗余和外观割裂问题
- **国家发明专利保护**：核心感应组件拥有国家发明专利（2020年授权），形成坚实的技术壁垒，保护客户ODM产品的市场独占性和差异化竞争力
- **双通道信号隔离**：自研双通道隔离算法确保接近感应信号与温控感应信号在电路层面和算法层面完全隔离，零干扰并行运行
- **紧凑一体化设计**：将两套感应系统集成于单一模块内，减少零部件数量和装配复杂度，提升系统整体可靠性和一致性
- **外观简洁美观**：仅需一个感应窗口，龙头表面更加整洁流畅，符合高端卫浴产品对工业美学的严苛要求
- **适配LED数显温控**：可搭配LED数显温控模组，实时反馈水温信息，提升用户交互体验和产品科技感
- **防水密封设计**：模块整体配合整机达到IP65防护等级，在潮湿浴室环境中长期稳定工作，适应商用和家用多种工况
- **低功耗长寿命**：采用低功耗感应方案，整机待机功耗极低，适配电池供电和低压供电场景，延长产品使用寿命
