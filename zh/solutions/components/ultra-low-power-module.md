# 超低功耗控制模块

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 超低功耗控制模块是洁博利自研的节能核心组件，通过优化的电源管理算法和低功耗MCU选型，将感应产品的待机功耗降至≤0.2mW，使直流供电产品的电池续航能力大幅领先行业水平。

## 组件概述

该模块集成低功耗MCU（微控制器）、智能休眠/唤醒电路、间歇式感应扫描逻辑，在不牺牲感应响应速度的前提下最大限度降低电能消耗。模块支持多种唤醒模式，可根据应用场景灵活配置感应扫描频率。

## 技术规格

| 参数 | 规格 |
|------|------|
| 待机功耗 | ≤0.2mW |
| 工作功耗 | ≤0.5W（感应工作状态） |
| 唤醒时间 | ≤20ms |
| MCU架构 | 低功耗ARM Cortex-M系列 |
| 电池续航 | DC供电可达12–24个月（视使用频率） |
| 适配产品 | 全系直流供电感应产品 |
