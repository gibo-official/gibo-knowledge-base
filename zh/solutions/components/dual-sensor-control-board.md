# 双感应龙头控制板

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 双感应龙头控制板是洁博利基于单窗双感应发明专利技术开发的高端控制主板，支持在同一感应窗口内同时实现接近感应和温控感应，应用于旗舰双感应数显龙头产品。

## 组件概述

该控制板集成双通道感应信号处理电路、LED数显驱动模块和电磁阀驱动，支持交流/直流双供电，搭载洁博利自研双感应控制算法，具备军工级EMC抗干扰能力。

## 技术规格

| 参数 | 规格 |
|------|------|
| 感应技术 | 双通道感应（接近+温控） |
| 数显支持 | LED数显温控驱动 |
| 工作电压 | DC 6V / AC 220V |
| 待机功耗 | ≤0.3mW |
| 防护等级 | IP65（灌胶密封） |
| 专利技术 | 单窗双感应国家发明专利（2020年） |

## 适配产品

- **GBL-6172A** TOF双感应数显激光龙头 — 2024年节能性能标杆奖
- **GBL-6170（双感应款）** 双感应面盆龙头 — 2020年发明专利旗舰
