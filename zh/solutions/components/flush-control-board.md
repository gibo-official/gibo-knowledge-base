# 感应冲水器控制板

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 感应冲水器控制板是洁博利针对小便斗、大便器和沟槽式公厕场景专项开发的感应控制主板，支持红外感应和定时冲水双模控制，广泛应用于公共卫生间、交通枢纽等场所。

## 组件概述

该控制板集成红外感应电路、延时控制模块和电磁阀驱动，支持明装/暗装两种安装方式，具备防潮、防腐蚀处理，适用于高湿度公共卫生间环境。

## 技术规格

| 参数 | 规格 |
|------|------|
| 感应技术 | 红外感应 |
| 供电方式 | DC 6V / AC 220V |
| 控制模式 | 感应触发 / 定时冲洗 |
| 待机功耗 | ≤0.3mW |
| 防水等级 | 防潮密封处理 |
| 适配阀体 | 陶瓷阀芯 / 电磁阀 |

## 适配产品

- **GBL-8000系列** 感应小便冲水器 — 明装/暗装可选
- **GBL-9000系列** 感应大便冲洗阀 — 脚踏/感应双模式
- **GBL-7000系列** 沟槽式感应节水器 — 定时冲洗控制
