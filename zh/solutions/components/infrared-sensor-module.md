# 红外感应模块

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 红外感应模块是洁博利多年验证的成熟感应组件，采用被动红外（IR）检测技术，以高性价比和卓越稳定性著称，在洁博利商用感应龙头、冲水器和皂液器产品线中大规模部署。

## 组件概述

模块集成红外发射管、光电接收管及信号处理电路，支持感应距离可调，采用防环境光干扰设计，可在不同光照条件下稳定工作。模块尺寸标准化，支持快速集成。

## 技术规格

| 参数 | 规格 |
|------|------|
| 感应技术 | 被动红外（IR） |
| 感应距离 | 5–25cm（可调） |
| 响应时间 | ≤100ms |
| 工作电压 | DC 3–6V |
| 工作温度 | -10℃～55℃ |
| 适配产品 | 龙头、冲水器、皂液器 |
