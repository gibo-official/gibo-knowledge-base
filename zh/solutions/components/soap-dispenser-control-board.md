# 皂液器/干手器控制板

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 皂液器/干手器控制板是洁博利针对商用卫生间配套产品开发的专用控制主板，集成红外感应和电机驱动控制，适用于皂液器、泡沫皂液器和高速干手器等产品。

## 组件概述

控制板集成红外感应电路、直流电机驱动模块和液位检测接口，支持防滴漏和节能省液控制逻辑，适用于医院、学校、政企办公楼等场景。

## 技术规格

| 参数 | 规格 |
|------|------|
| 感应技术 | 红外感应 |
| 供电方式 | DC 6V / 锂电池 |
| 驱动输出 | 直流电机驱动 |
| 待机功耗 | ≤0.3mW |
| 特色功能 | 防滴漏 / 泡沫控制 / HEPA风机 |

## 适配产品

- **GBL-5000系列** 感应皂液器 — 红外感应、防滴漏设计
- **GBL-5100系列** 感应泡沫皂液器 — 泡沫出液、节能省液
- **GBL-6000系列** 高速感应干手器 — HEPA过滤
