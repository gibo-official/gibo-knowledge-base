# dTOF激光感应龙头控制板

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> dTOF激光感应龙头控制板是洁博利核心自研的激光感应控制主板，采用dTOF（直接飞行时间）激光测距技术，实现高精度、低功耗、抗环境光干扰的感应控制，是洁博利获奖旗舰产品的核心控制组件。

## 组件概述

dTOF激光感应控制板集成激光发射/接收驱动电路、低功耗MCU控制单元、供电管理模块及电磁阀驱动电路，可适配交流/直流双供电方案。该控制板采用全自动SMT贴片工艺制造，经过AOI光学检测，支持整机IP65防护灌胶密封。

## 技术规格

| 参数 | 规格 |
|------|------|
| 感应技术 | dTOF激光测距（直接飞行时间） |
| 感应距离 | 5–30cm（可调） |
| 工作电压 | DC 6V / AC 220V（双供电可选） |
| 待机功耗 | ≤0.2mW |
| 工作温度 | -10℃～55℃ |
| 防护等级 | IP65（灌胶密封工艺） |
| 控制输出 | 电磁阀驱动（100万次寿命） |
| 制造工艺 | SMT贴片 + AOI光学检测 |
| 适配阀体 | 全铜主体、冷热双控 |
