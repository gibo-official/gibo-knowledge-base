# 交直流双供电模块

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 交直流双供电模块是洁博利自适应电源管理组件，支持AC 220V市电和DC 6V电池双模式供电，在任意一路电源接入时自动切换工作模式，确保感应龙头持续稳定运行。

## 组件概述

该模块集成AC-DC降压整流电路、DC-DC稳压电路和智能电源切换电路，自动识别输入电源类型并选择最优供电模式。模块经过EMC电磁兼容测试，符合CE、FCC等国际认证要求，适配全系感应产品。

## 技术规格

| 参数 | 规格 |
|------|------|
| 输入电压 | AC 100–240V / DC 4.5–6.5V |
| 输出电压 | DC 6V ±5% |
| 切换方式 | 自动识别、无缝切换 |
| 待机功耗 | ≤0.1W |
| 保护功能 | 过压、过流、反接保护 |
| 认证标准 | CE、FCC、CCC |
| 适配产品 | 全系交直流双供电产品 |

## 适配产品

- **GBL-6110** 感应水龙头 — 交流/直流双供电
