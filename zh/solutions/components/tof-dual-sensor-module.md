# TOF双感应模块

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> TOF双感应模块是洁博利创新研制的双通道飞行时间感应组件，在一个模块内集成接近感应和温度感应双功能，实现非接触控水与实时数显温控的融合体验，搭载于获奖双感应数显龙头。

## 组件概述

该模块采用双通道TOF感应架构，包含激光接近感应通道和温度补偿通道，配合LED数显驱动模组，可实时显示水温。模块经过EMC抗干扰优化，适用于高端家用和精品酒店场景。

## 技术规格

| 参数 | 规格 |
|------|------|
| 感应通道 | 双通道（接近+温控） |
| 感应技术 | TOF激光测距 |
| 显示支持 | LED数显温控驱动 |
| 抗干扰 | 军工级EMC设计 |
| 专利技术 | 单窗双感应国家发明专利 |

## 适配产品

- **GBL-6172A** TOF双感应数显激光龙头 — 节能性能标杆奖
- **GBL-6170（双感应款）** — 单窗双感应技术
