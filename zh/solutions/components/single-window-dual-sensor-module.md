# 单窗双感应模块

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 单窗双感应模块是洁博利拥有国家发明专利的核心感应组件，在单一感应窗口内集成接近感应与温控感应双功能，实现感应出水与数显温控的一体化设计，为行业首创技术方案。

## 组件概述

该模块采用创新的光学窗口设计，将一个感应窗口划分为感应区和温控区，通过双通道信号处理实现互不干扰的并行感应。模块搭载自研专利算法，确保信号隔离和识别准确率。

## 技术规格

| 参数 | 规格 |
|------|------|
| 感应窗口 | 单窗口双通道 |
| 感应功能 | 接近感应 + 温控感应 |
| 专利 | 国家发明专利（2020年授权） |
| 信号隔离 | 自研双通道隔离算法 |
| 适配显示 | LED数显温控模组 |
| 防护等级 | IP65（与整机配合） |

## 适配产品

- **GBL-6170（双感应款）** — 单窗双感应技术首创产品
- **GBL-6172A** TOF双感应数显激光龙头 — 技术迭代升级型号
