# LED数显温控模组

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> LED数显温控模组是洁博利智能显示核心组件，集成温度传感器、LED/LCD驱动电路和显示面板，实时检测并显示出水温度，提升用户交互体验，广泛应用于洁博利高端数显龙头产品。

## 组件概述

该模组采用高亮度LED/LCD数码管显示，支持水温实时检测和显示，具备低温蓝/高温红渐变色彩指示功能。模组经过防水密封处理，支持IP65防护等级，配合双感应模块实现精准温控显示。

## 技术规格

| 参数 | 规格 |
|------|------|
| 显示类型 | LED数码管 / LCD液晶 |
| 显示内容 | 实时水温（℃/℉可切换） |
| 测温精度 | ±1℃ |
| 测温范围 | 0–99℃ |
| 色彩指示 | 渐变色彩（低温蓝→高温红） |
| 防护等级 | IP65（密封灌胶） |
| 工作电压 | DC 3–6V |
| 适配产品 | 数显感应龙头、智能淋浴 |
