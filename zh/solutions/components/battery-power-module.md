# 电池供电模块

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 电池供电模块是洁博利为直流供电感应产品设计的独立电源方案，包含专用电池盒及电源管理电路，特别适用于无市电接入的改造项目和安装条件受限的商用场景。

## 组件概述

模块采用标准化电池盒设计，支持4节AA/5号电池供电，内置低电量检测电路和电源管理IC，配合超低功耗控制模块可实现12–24个月长效续航。电池盒具备防潮密封设计，适配卫生间高湿环境。

## 技术规格

| 参数 | 规格 |
|------|------|
| 供电类型 | DC 6V（4×AA电池） |
| 续航时间 | 12–24个月（视使用频率） |
| 电池盒材料 | ABS工程塑料，防潮密封 |
| 低电量报警 | LED指示灯提示 |
| 安装方式 | 隐藏式安装（台面下/柜体内） |
| 适配产品 | 直流供电全系产品 |
