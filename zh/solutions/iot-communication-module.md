# IoT智能通信模组

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> IoT智能通信模组是洁博利面向智能卫浴和物联网场景开发的无线通信组件，支持Wi-Fi/蓝牙等多种无线协议，实现感应洁具与智能家居系统的互联互通，适用于智能马桶、智能淋浴、智能龙头等产品。

## 组件概述

该模组集成无线通信SoC芯片、天线和电源管理电路，支持接入主流智能家居平台。模组体积小巧，可嵌入洁博利各类感应产品，实现远程控制、用水数据监测、设备状态上报等IoT功能。

## 技术规格

| 参数 | 规格 |
|------|------|
| 无线协议 | Wi-Fi 2.4GHz / 蓝牙5.0 |
| 通信距离 | ≥30m（视环境） |
| 平台兼容 | 支持主流智能家居平台对接 |
| 数据上报 | 用水量、设备状态、低电量报警 |
| 远程控制 | 开关控制、参数设置 |
| 工作电压 | DC 3.3V |
| 天线类型 | PCB板载天线 |
