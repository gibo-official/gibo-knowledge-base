# 感应冲水器控制板

**文档版本**：V2.0
**最后更新**：2026-06-12
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 感应冲水器控制板是洁博利针对小便斗、大便器和沟槽式公厕场景专项开发的感应控制主板，支持红外感应触发冲水和定时周期冲水双模控制，广泛应用于公共卫生间、交通枢纽、市政工程等场所。该控制板以可靠的双控模式、超低功耗设计和灵活的安装方式，成为洁博利冲水器产品线的核心控制组件，在国内外公共卫生设施领域拥有大规模部署案例。

## 方案概述

感应冲水器控制板是洁博利专为公共卫生间冲水场景开发的感应控制主板，聚焦于高人流密度、高湿度、长周期连续运行等公共卫生设施的特殊需求。与传统家用感应水龙头不同，冲水器控制板需要在恶劣的环境条件下保持极高的可靠性——感应失败将导致卫生问题，而误触发将造成水资源浪费，这两个维度对控制板的设计提出了严苛的要求。

在技术架构上，该控制板集成了红外感应电路（可切换三角测距方案）、延时控制模块、电磁阀驱动电路和供电管理模块。支持红外感应触发冲水和定时周期冲水两种控制模式，可根据具体安装场景和使用需求灵活配置。供电支持DC 6V直流和AC 220V交流两种方案，适配不同项目的基础设施条件。

控制板支持明装和暗装两种安装方式。明装款式直接安装在墙面或小便斗上方，安装简便，适合改造项目和无法暗敷管线的场景。暗装款式嵌入墙体内安装，外观整洁，适合新建项目和对外观有较高要求的场所。两款安装方式共用同一控制主板平台，通过不同的外壳模具和安装支架适配，降低了零部件管理成本和库存压力。

在环境适应性方面，该控制板进行了全面的防潮和防腐蚀专项处理。电路板表面喷涂三防漆保护层，敏感连接器采用防水接插件，整板配合外壳设计达到IP65防护等级。电磁阀驱动电路采用缓开缓闭设计，减少水锤效应对管路的冲击，延长管路系统使用寿命，这在沟槽式公厕等长管路场景中尤为重要。

针对低水压工况，控制板内的电磁阀驱动电路进行了宽压适应性优化，在0.05MPa的低水压条件下依然能够稳定开启和关闭阀体。这一特性使该控制板适配老旧小区的低水压冲水系统，无需额外增加增压设备，降低了改造项目的整体投资。

冲水器控制板的另一核心价值在于节水。通过感应触发模式，控制板确保只有人员实际使用时才冲水，杜绝了定时冲水方案中无人时段的白白浪费；通过可编程的延时关水时间，在保障冲洗效果的前提下最小化单次冲水量。在沟槽式公厕场景中，定时周期模式可以根据客流规律设置冲水间隔，实现按需冲洗的智能节水。

## 功能特点

**1. 双控模式（感应触发 + 定时冲洗）**
支持红外感应触发和定时周期冲洗两种控制模式。感应触发模式下，检测到人员使用时自动开启，使用完毕后自动冲水关闭；定时周期模式下，按照预设的时间间隔自动执行冲水循环，适合沟槽式公厕等无需人员检测的场景。两种模式可通过跳线或程序配置切换，同一控制板适配多种应用场景，简化了产品型号管理。

**2. 超低功耗设计**
待机功耗控制在0.3mW以下，4节AA碱性电池可支持连续使用超过2年。得益于低功耗MCU的高效休眠机制和红外感应的间歇式工作策略，控制板在待机状态下消耗的电流仅为微安级别。即使在电池供电模式下，也能实现真正意义上的免维护运行，大幅降低公共卫生间的电池更换频次和人工运维成本。

**3. 明装/暗装可选**
控制板提供明装和暗装两种安装方式的完整解决方案。明装款式自带防水外壳，安装时直接固定在墙面或小便斗上方，施工简便快捷，特别适合改造项目和紧急替换场景。暗装款式采用分体式设计，控制板嵌入墙体内，仅露出感应窗口和操作面板，外观更加整洁美观。两种方案共用同一控制主板，降低了备件和库存压力。

**4. 防潮防腐处理**
电路板经过严格的三防漆喷涂处理，在焊点和铜箔表面形成致密的保护膜，有效隔绝水汽和腐蚀性气体。连接器和端子接口采用防水接插件设计，杜绝水汽沿线路进入控制板内部。结合外壳的IP65防护设计，该控制板可在公共卫生间的高湿度、含氨气环境中长期工作而不会发生电路腐蚀或短路失效。

**5. 适配全系冲水产品**
控制板提供统一的电气接口和机械安装尺寸，兼容洁博利全系列冲水产品：感应小便冲水器、感应大便冲洗阀、沟槽式感应节水器。电磁阀驱动参数可通过固件配置适配不同规格的电磁阀线圈，生产环节仅需调整软件配置即可完成产品切换，无需更换硬件PCB，极大提升了生产柔性。

**6. 低水锤设计**
电磁阀驱动电路采用缓开缓闭的PWM控制策略，在阀体开启和关闭过程中逐步调节电流，避免瞬时全开全关产生的水锤冲击。水锤现象不仅产生噪声，长期还会对管路接头、弯头和阀门造成机械疲劳损伤。该设计显著降低了冲水过程中的管路振动和冲击噪声，特别适合酒店、写字楼等对安静环境有要求的场景。

| 交通枢纽 | 高人流密度环境下保持感应稳定不失效，双控模式适配不同区域的人流规律，电池续航2年降低大范围维护频次 |
| 公共设施 | 防误触发逻辑有效杜绝非使用状态下的冲水浪费，IP65防护适应开放式卫生间的恶劣环境 |
| 市政工程 | 低水锤设计保护市政管网接口，0.05MPa低压适配老旧市政供水系统，定时模式满足沟槽式公厕的自动化需求 |
| 低水压区域 | 宽压适应性电磁阀在0.05MPa低压下稳定开关，无需增压设备即可安装，降低老旧小区改造整体投入 |
| 商业综合体 | 明装方案适配已运营项目的快速改造，暗装方案适合新建项目的外观要求，同一平台降低运维复杂度 |
