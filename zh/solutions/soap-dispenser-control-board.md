# 皂液器/干手器控制板

**文档版本**：V2.0
**最后更新**：2026-06-12
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 皂液器/干手器控制板是洁博利针对商用卫生间配套产品开发的专用控制主板，集成红外感应电路、直流电机驱动模块和液位检测接口，支持防滴漏和节能省液控制逻辑，适用于医院、学校、政企办公楼等场景，适配感应皂液器、泡沫皂液器和高速干手器。

## 方案概述

皂液器/干手器控制板是洁博利面向商用卫生间配套市场推出的专用控制主板方案。该方案基于成熟的红外感应控制技术，集成红外发射与接收电路、直流电机驱动模块和液位检测接口，配合洁博利自研Liteon灵动感应算法，实现精准的非接触感应出液控制。控制板的核心控制逻辑围绕"防滴漏"和"节能省液"两大目标设计：防滴漏逻辑在出液结束时自动执行反向抽吸动作，将残留在出液口的液体抽回管路内部，杜绝滴漏污染台面；节能省液逻辑通过精准控制电机运转时长和出液量，在保证足够清洁用量的前提下最大限度减少浪费，与传统按压式皂液器相比可节省30%以上的皂液消耗。控制板支持感应距离多档可调和出液量分级设定，ODM客户可根据不同使用场景灵活配置。针对泡沫皂液器，控制板内置泡沫发生控制逻辑，通过精确控制气液混合比和泡沫浓度，产生绵密细腻的泡沫，提升洗手体验。针对高速干手器，控制板集成了HEPA过滤风机驱动模块，可驱动高速无刷电机并监控HEPA滤网运行状态，确保出风洁净卫生。此外，控制板还支持锂电池充电管理功能，适配内置锂电池供电方案，满足无电源安装场景的需求。该控制板采用模块化设计，同一硬件平台通过固件配置即可适配皂液器、泡沫皂液器、干手器三种产品类型，大幅降低ODM客户的库存管理和开发成本。

**方案定位**：商用卫生间皂液器/干手器专用控制方案
**核心技术**：IR红外感应技术 + Liteon灵动感应算法（核心技术#7）+ 低功耗多稳态灵动感应技术（核心技术#6）
**供电方式**：DC 6V / 锂电池双供电
**适用场景**：医院、学校、政企办公楼、商场等公共卫生间

## 功能特点

- **红外感应控制**：基于成熟红外反射式感应技术，搭载Liteon灵动感应算法，自适应环境光线强度调节检测阈值，精准识别手部接近动作，有效过滤水滴、飞虫等微小干扰信号，杜绝误触发。
- **防滴漏设计**：出液结束时自动执行反向抽吸动作，将出液口残留液体抽回，杜绝滴漏污染台面，保持洗手台卫生整洁。该设计特别适用于医院、食品处理区等对卫生要求严格的场景。
- **泡沫控制技术**：通过精确控制气液混合电机转速比，自动生成绵密细腻的泡沫，泡沫直径均匀、附着力强，洗手时不易滑落。单次出泡量可调，泡沫浓度可软件配置，适配不同品牌皂液。
- **节能省液逻辑**：精准控制电机运转时长实现定量出液，避免过量出液浪费。与传统按压式皂液器相比，皂液使用量节省30%以上，大容量皂液瓶（500ml）单次补充可用数千次，大幅降低运维成本。
- **液位检测**：内置电容式或浮子式液位检测接口，实时监测皂液余量。液位低于设定阈值时自动触发报警信号（LED闪烁或蜂鸣），提醒运维人员及时补充皂液，避免空置使用影响用户体验。液位过低时自动停机保护电机。
- **HEPA过滤（干手器）**：干手器专用版集成高速无刷电机驱动模块，支持PWM调速控制，配合HEPA高效过滤网，有效过滤空气中99.97%的0.3μm颗粒物，确保出风洁净卫生，特别适用于医院、食品加工等对空气质量有严格要求的场景。
