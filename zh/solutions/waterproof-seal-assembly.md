# IP65防水密封组件

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> IP65防水密封组件是洁博利确保感应产品在卫生间高湿环境下长期可靠运行的核心防护方案，采用灌胶密封工艺配合多重密封结构，实现整机IP65防护等级，有效防止水汽、灰尘侵入电子模块。

## 组件概述

密封组件包含电子模块灌胶密封层、按键/显示区域密封圈、控制板三防漆涂层和壳体防水结构。每台产品出厂前经过严格的气密性测试，确保在长期高湿度环境中电子模块零故障。

## 技术规格

| 参数 | 规格 |
|------|------|
| 防护等级 | IP65（全机） |
| 密封工艺 | 电子模块灌胶密封 |
| 覆盖涂层 | 三防漆（防潮/防盐雾/防霉） |
| 密封件 | 硅胶密封圈 / 防水按键 |
| 测试标准 | 喷水测试（6.3mm喷嘴，12.5L/min） |
| 适用环境 | 卫生间、厨房等高湿场景 |
