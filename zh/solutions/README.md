### A2. 三角测距式蹲便器感应控制模块

#### 方案概述

三角测距式蹲便器感应控制模块是洁博利为解决公共卫生间蹲便器自动冲水难题而自主研发的专属方案。传统红外感应方案在蹲便器隔间场景中存在诸多痛点：用户身体遮挡导致红外信号无法穿透隔板、隔板反光引起红外误判、人离开后未能及时触发冲水等。该方案采用三角测距感应技术，通过发射器与接收器之间的固定几何夹角，利用三角测量原理精确计算出目标的距离和位置信息，从根本上解决感应精度问题。该方案经过三年多研发与试验，已实现量产市场落地，在高铁站、机场、医院等高频使用场景中经过充分验证。方案搭载60μA超低功耗电路设计，确保电池供电产品保持长续航运行，同时配备低水锤缓闭冲水阀，有效保护管路系统。

**方案定位**：公共卫浴蹲便器专属方案
**核心技术**：三角测距感应技术（核心技术#1）、低功耗多稳态灵动感应技术（核心技术#6）、电磁阀低水锤设计技术（核心技术#15）

### A3. 马桶dTOF挥手脚踢激光感应冲水控制模块

#### 方案概述

马桶dTOF挥手脚踢激光感应冲水控制模块是洁博利面向高端智能马桶市场推出的专属冲水控制方案。该方案采用dTOF（Direct Time-of-Flight，直接飞行时间）激光测距感应技术，通过发射940nm VCSEL激光脉冲并精确测量光子飞行时间，直接计算出目标与传感器之间的距离，精度达到毫米级。与传统红外感应依赖反射光强判断目标存在不同，dTOF技术不受目标颜色、材质和表面反射率的影响，黑色衣物、白色瓷砖等低反射率场景下依然稳定检测。该方案创新性地集成了挥手感应和脚踢感应双触发模式，用户可根据使用习惯自由选择触发方式。方案支持分段冲水逻辑，可根据感应时长自动识别大小冲水量，兼顾节水效果和冲洗效果。该方案可适配普通马桶智能化改装，也可集成至一体式智能马桶中，是传统马桶智能化升级的理想方案。
