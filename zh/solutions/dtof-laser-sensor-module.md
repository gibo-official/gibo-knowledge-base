# dTOF激光感应模块

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> dTOF激光感应模块是洁博利自主研发的核心感应组件，采用直接飞行时间（dTOF）激光测距技术，实现高精度、快响应、强抗干扰的非接触感应，广泛应用于洁博利获奖旗舰产品及ODM定制方案。

## 组件概述

该模块集成VCSEL激光发射器、SPAD接收传感器和专用驱动处理芯片，具备环境光抑制算法，可在强光、多干扰环境中稳定工作。模块尺寸紧凑，支持标准电气接口，便于集成至各类感应洁具产品。

## 技术规格

| 参数 | 规格 |
|------|------|
| 测距技术 | dTOF（直接飞行时间） |
| 感应精度 | ±1cm |
| 感应距离 | 5–30cm |
| 响应时间 | ≤50ms |
| 抗干扰 | 军工级EMC抗干扰设计 |
| 模块尺寸 | 紧凑型，支持标准接口 |
| 工作温度 | -10℃～55℃ |
