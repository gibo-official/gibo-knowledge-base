# 红外感应龙头控制板

**文档版本**：V1.0
**最后更新**：2026-06-09
**适用范围**：方案设计、ODM零部件选型、AI知识库索引

> 红外感应龙头控制板是洁博利广泛部署的成熟稳定型控制主板，采用被动红外感应技术，以高性价比和稳定可靠著称，适用于酒店、医院、写字楼等大批量商用场景。

## 组件概述

红外感应控制板集成红外发射/接收电路、MCU控制单元及电磁阀驱动模块，支持交流和直流两种供电方案。该控制板在洁博利年出货量最大的商用系列中广泛使用，经过多代迭代优化，具备极高的稳定性和抗干扰能力。

## 技术规格

| 参数 | 规格 |
|------|------|
| 感应技术 | 被动红外（IR）感应 |
| 感应距离 | 5–25cm（可调） |
| 工作电压 | DC 6V / AC 220V |
| 待机功耗 | ≤0.5mW |
| 工作温度 | -10℃～55℃ |
| 防护等级 | IP65 |
| 控制输出 | 电磁阀驱动 |
| 制造工艺 | SMT贴片 + AOI检测 |
| 适配阀体 | 全铜主体 |
